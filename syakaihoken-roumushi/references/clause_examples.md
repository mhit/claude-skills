## 総則

### 目的

**基本パターン**
```
第1条（目的）
この就業規則（以下「規則」という。）は、労働基準法（昭和22年法律第49号）
第89条に基づき、株式会社○○（以下「会社」という。）の労働者の労働条件、
服務規律その他の就業に関する事項を定めるものである。
```

**詳細版（企業理念を含む）**
```
第1条（目的）
1. この就業規則（以下「規則」という。）は、株式会社○○（以下「会社」という。）
   における労働者の労働条件、服務規律その他の就業に関する事項を定め、
   会社と労働者が相互の信頼関係の下に、協力して業務を遂行し、
   もって会社の発展と労働者の生活の向上を図ることを目的とする。
2. 労働者は、この規則を遵守し、誠実に職務を遂行しなければならない。
```

### 適用範囲

**正社員のみの場合**
```
第2条（適用範囲）
この規則は、会社に雇用されるすべての労働者に適用する。
```

**正社員とパートを区別する場合**
```
第2条（適用範囲）
1. この規則は、会社の正社員に適用する。
2. パートタイマー、アルバイト等の労働者については、別に定めるパートタイマー
   就業規則による。ただし、パートタイマー就業規則に定めのない事項については、
   この規則を適用する。
```

## 採用・異動

### 採用手続

**基本パターン**
```
第○条（採用手続）
1. 会社は、入社を希望する者の中から選考試験を行い、これに合格した者を
   採用する。
2. 前項の選考に当たっては、履歴書その他会社が指定する書類を提出させ、
   面接試験を行う。
```

**詳細版**
```
第○条（採用手続）
1. 会社は、入社を希望する者の中から選考試験（書類選考、筆記試験、
   面接試験、健康診断等）を行い、これに合格した者を採用する。
2. 前項の選考に当たっては、次の書類を提出させる。
   (1) 履歴書（写真貼付）
   (2) 職務経歴書
   (3) 卒業証明書又は卒業見込証明書
   (4) 健康診断書（3ヶ月以内に受診したもの）
   (5) その他会社が指定する書類
3. 採用が決定した者については、速やかに労働条件通知書を交付する。
```

### 試用期間

**標準パターン（3ヶ月）**
```
第○条（試用期間）
1. 新たに採用した者については、採用の日から3ヶ月間を試用期間とする。
   ただし、会社が特に認めた者については、この期間を短縮し、又は設けない
   ことがある。
2. 試用期間中又は試用期間満了時に、労働者として不適格と認めた場合は、
   本採用を行わず、又は本採用後であっても解雇することがある。
3. 試用期間は、勤続年数に通算する。
```

**IT業種向け（試用期間中の評価基準明示）**
```
第○条（試用期間）
1. 新たに採用した者については、採用の日から3ヶ月間を試用期間とする。
2. 試用期間中は、次の事項について評価を行う。
   (1) 勤務態度及び勤怠状況
   (2) 業務遂行能力及び技術力
   (3) 協調性及びコミュニケーション能力
3. 前項の評価の結果、労働者として不適格と認めた場合は、本採用を行わず、
   又は本採用後であっても14日前に予告して解雇することがある。
```

### 人事異動

**基本パターン**
```
第○条（人事異動）
1. 会社は、業務上必要がある場合は、労働者に対して配置転換、転勤、
   出向又は職種の変更を命ずることがある。
2. 労働者は、正当な理由なく前項の命令を拒むことはできない。
```

**配慮規定付き**
```
第○条（人事異動）
1. 会社は、業務上必要がある場合は、労働者に対して配置転換、転勤、
   出向又は職種の変更を命ずることがある。
2. 会社は、前項の命令を行うに当たっては、労働者の個人的事情に可能な
   限り配慮するものとする。
3. 労働者は、正当な理由なく第1項の命令を拒むことはできない。
```

**育児・介護配慮版**
```
第○条（人事異動）
1. 会社は、業務上必要がある場合は、労働者に対して配置転換、転勤、
   出向又は職種の変更を命ずることがある。
2. 会社は、育児又は介護を行う労働者について配置転換又は転勤を命ずる
   場合には、当該労働者の子の養育又は家族の介護の状況に配慮するものとする。
3. 労働者は、正当な理由なく第1項の命令を拒むことはできない。
```

## 労働時間・休憩・休日

### 労働時間（通常の事業場）

**固定労働時間制**
```
第○条（労働時間）
1. 労働時間は、1週間については40時間、1日については8時間とする。
2. 始業・終業の時刻及び休憩時間は、次のとおりとする。
   始業時刻  午前9時00分
   終業時刻  午後6時00分
   休憩時間  正午から午後1時まで
3. 前項の規定にかかわらず、業務の都合その他やむを得ない事情により、
   始業及び終業の時刻並びに休憩時間を繰り上げ又は繰り下げることがある。
   この場合、前日までに労働者に通知する。
```

### 労働時間（IT業種・フレックスタイム制）

**フレックスタイム制**
```
第○条（フレックスタイム制）
1. 労働者は、1ヶ月の清算期間における総労働時間の範囲内で、始業及び
   終業の時刻を自主的に決定することができる。
2. 清算期間における総労働時間は、○○時間とする。
3. 標準となる1日の労働時間は、8時間とする。
4. 労働者は、次のコアタイムには勤務しなければならない。ただし、
   会社が承認した場合は、この限りでない。
   午前10時00分から午後3時00分まで（休憩時間を除く）
5. フレキシブルタイムは、次のとおりとする。
   午前7時00分から午前10時00分まで
   午後3時00分から午後10時00分まで
```

**スーパーフレックス（コアタイムなし）**
```
第○条（フレックスタイム制）
1. 会社は、労働者に対し、1ヶ月の清算期間における総労働時間の範囲内で、
   始業及び終業の時刻を労働者の決定に委ねるフレックスタイム制を適用する。
2. 清算期間は、毎月1日を起算日とする1ヶ月間とする。
3. 清算期間における総労働時間は、○○時間とする。
4. 標準となる1日の労働時間は、8時間とする。
5. フレキシブルタイムは、午前5時00分から午後10時00分までとし、
   この時間帯内で労働者が自由に始業及び終業の時刻を決定できる。
6. 労働者は、業務の都合上必要がある場合を除き、勤務日の前日までに、
   翌日の勤務予定を所属長に報告するものとする。
```

### 休日

**完全週休2日制**
```
第○条（休日）
1. 休日は、次のとおりとする。
   (1) 土曜日及び日曜日
   (2) 国民の祝日（日曜日と重なったときは翌日）
   (3) 年末年始（12月29日から1月3日まで）
   (4) その他会社が指定する日
2. 業務の都合により、前項の休日を他の日に振り替えることがある。
   この場合、前日までに振替日を指定して労働者に通知する。
```

**シフト制（サービス業）**
```
第○条（休日）
1. 休日は、1週間につき1日以上とし、4週間を通じて4日以上とする。
2. 休日は、勤務割表により各労働者ごとに指定する。
3. 勤務割表は、毎月○日までに翌月分を作成し、労働者に通知する。
```

### 時間外労働・休日労働

**基本パターン**
```
第○条（時間外労働及び休日労働）
1. 業務の都合により、第○条の所定労働時間を超え、又は第○条の所定休日に
   労働させることがある。
2. 前項の場合、労働基準法第36条に基づく労使協定の範囲内で、必要最小限の
   時間外労働又は休日労働を命ずることがある。
3. 妊娠中の女性及び産後1年を経過しない女性（以下「妊産婦」という。）が
   請求した場合には、時間外労働、休日労働又は深夜業をさせない。
```

**事前申請制**
```
第○条（時間外労働及び休日労働）
1. 業務の都合により、第○条の所定労働時間を超え、又は第○条の所定休日に
   労働させることがある。
2. 時間外労働又は休日労働を行う場合は、原則として事前に所属長の許可を
   得なければならない。ただし、緊急やむを得ない場合は、事後速やかに
   報告するものとする。
3. 会社は、労働者の健康を考慮し、時間外労働及び休日労働の削減に努める。
```

## 賃金

### 賃金の構成

**基本パターン**
```
第○条（賃金）
労働者の賃金は、次のとおりとする。
(1) 基本給
(2) 諸手当
   ア 職務手当
   イ 通勤手当
   ウ 家族手当
   エ 住宅手当
   オ その他会社が定める手当
```

**詳細版（IT業種向け）**
```
第○条（賃金の構成）
労働者の賃金は、次のとおりとする。
(1) 基本給（月給）
(2) 職務手当（役職に応じて支給）
(3) 技術手当（保有資格・技術レベルに応じて支給）
(4) プロジェクト手当（プロジェクト業務に従事する場合に支給）
(5) 通勤手当（実費支給、上限月額○○円）
(6) 時間外勤務手当
(7) 休日勤務手当
(8) 深夜勤務手当
```

### 賃金の支払

**基本パターン**
```
第○条（賃金の支払）
1. 賃金は、毎月末日に締め切り、翌月25日に労働者の指定する銀行口座に
   振り込むことにより支払う。ただし、支払日が休日に当たる場合は、
   その前日に支払う。
2. 賃金は通貨で直接労働者に、その全額を支払う。ただし、次の各号に
   掲げるものは、賃金から控除する。
   (1) 源泉所得税
   (2) 住民税
   (3) 健康保険料、介護保険料、厚生年金保険料及び雇用保険料の労働者負担分
   (4) その他法令に基づくもの
   (5) 労使協定に基づくもの
```

**デジタル払い対応版（2023年改正対応）**
```
第○条（賃金の支払方法）
1. 賃金は、毎月末日に締め切り、翌月25日に次のいずれかの方法により
   支払う。ただし、支払日が休日に当たる場合は、その前日に支払う。
   (1) 労働者の指定する銀行口座への振込
   (2) 労働者の同意を得た場合、厚生労働大臣が指定する資金移動業者の
       口座への資金移動
2. 前項第2号の方法による支払いを希望する労働者は、所定の同意書を
   提出しなければならない。
3. 賃金は通貨で直接労働者に、その全額を支払う。ただし、次の各号に
   掲げるものは、賃金から控除する。
   (以下、控除項目を列挙)
```

### 割増賃金

**2023年改正対応（中小企業の月60時間超50%）**
```
第○条(割増賃金)
1. 時間外労働、休日労働及び深夜労働に対する割増賃金は、次の割増賃金率
   により計算した額を支払う。
   (1) 時間外労働
      ア 1ヶ月の時間外労働が60時間以下の場合: 25%以上
      イ 1ヶ月の時間外労働が60時間を超えた場合:
         60時間までは25%以上、60時間を超えた部分は50%以上
   (2) 法定休日労働: 35%以上
   (3) 深夜労働（午後10時から午前5時まで）: 25%以上
2. (1)のア又はイと(3)が重複する場合は、それぞれの割増率を合計した率
   （50%又は75%）で計算する。
3. (2)と(3)が重複する場合は、それぞれの割増率を合計した率（60%）で
   計算する。
```

## 休暇

### 年次有給休暇

**年5日取得義務対応版**
```
第○条（年次有給休暇）
1. 入社日から起算して6ヶ月間継続勤務し、全労働日の8割以上出勤した
   労働者に対しては、10日の年次有給休暇を付与する。その後は1年ごとに
   所定の日数を付与する。
2. 年次有給休暇は、労働者の請求する時季に与える。ただし、事業の正常な
   運営を妨げる場合は、他の時季に変更することがある。
3. 前項の規定にかかわらず、労働者代表との協定により、各労働者の年次
   有給休暇のうち5日を超える部分について、あらかじめ時季を指定して
   与えることがある。
4. 第1項及び第2項の規定にかかわらず、年10日以上の年次有給休暇が
   付与される労働者については、付与日から1年以内に、当該労働者の
   有する年次有給休暇日数のうち5日について、会社が労働者の意見を
   聴取し、その意見を尊重した上で、あらかじめ時季を指定して与える。
   ただし、労働者が第2項又は第3項の規定による年次有給休暇を取得した
   場合においては、当該取得した日数分を5日から控除するものとする。
```

### 特別休暇（慶弔休暇）

**基本パターン**
```
第○条（特別休暇）
労働者が次の各号のいずれかに該当する場合は、特別休暇を与える。
この場合、特別休暇は有給とする。
(1) 本人が結婚するとき: ○日
(2) 子が結婚するとき: ○日
(3) 配偶者、子又は父母が死亡したとき: ○日
(4) 兄弟姉妹、祖父母、配偶者の父母が死亡したとき: ○日
(5) 妻が出産するとき: ○日
```

## 服務規律

### 基本的服務規律

**標準パターン**
```
第○条（服務規律）
労働者は、次の事項を守らなければならない。
(1) 許可なく職務以外の目的で会社の施設、物品等を使用しないこと
(2) 職務に関連して自己の利益を図り、又は他より不当に金品を借用し、
    若しくは贈与を受ける等不正な行為を行わないこと
(3) 勤務中は職務に専念し、正当な理由なく勤務場所を離れないこと
(4) 会社の名誉や信用を損なう行為をしないこと
(5) 在職中及び退職後においても、業務上知り得た会社、取引先等の
    機密を漏らさないこと
(6) 互いに人格を尊重し、差別的言動、暴力行為、誹謗中傷等を行わないこと
(7) 酒気を帯びて就業しないこと
(8) その他労働者としてふさわしくない行為をしないこと
```

### ハラスメント防止

**2020年改正対応（パワハラ防止法対応）**
```
第○条（ハラスメントの禁止）
1. すべての労働者は、職場において、次に掲げるハラスメント行為を
   行ってはならない。
   (1) パワーハラスメント
       優越的な関係を背景とした言動であって、業務上必要かつ相当な
       範囲を超えたものにより、他の労働者の就業環境を害すること
   (2) セクシュアルハラスメント
       職場において行われる性的な言動に対する他の労働者の対応により
       当該労働者がその労働条件について不利益を受けること、又は
       性的な言動により他の労働者の就業環境が害されること
   (3) マタニティハラスメント
       職場において行われる妊娠、出産、育児休業等に関する言動により
       他の労働者の就業環境が害されること
2. 会社は、ハラスメント行為を確認した場合、行為者に対し懲戒処分を
   含む厳正な対処を行う。
3. ハラスメントに関する相談窓口は、人事部（又は総務部）とする。
   窓口の連絡先は別途掲示する。
4. 会社は、相談者及び行為者等のプライバシーを保護するとともに、
   相談したこと又は事実関係の確認に協力したこと等を理由として
   不利益な取扱いを行わない。
```

### テレワーク

**テレワーク規定**
```
第○条（テレワーク）
1. 会社は、業務上必要があると認める場合、労働者の自宅その他会社が
   認めた場所において、情報通信機器を利用して業務に従事させることが
   ある（以下「テレワーク」という。）。
2. テレワークの実施については、別に定めるテレワーク規程による。
3. テレワークを実施する労働者の労働時間、賃金その他の労働条件については、
   この規則の定めるところによる。ただし、テレワーク規程に別段の定めが
   ある場合は、その定めによる。
```

### 副業・兼業

**原則許可制**
```
第○条（副業・兼業）
1. 労働者は、会社の事前の許可を受けた場合に限り、勤務時間外において
   他の会社等の業務に従事することができる。
2. 労働者は、前項の許可を受けようとするときは、所定の届出書に必要事項を
   記載の上、所属長を経由して会社に提出しなければならない。
3. 会社は、第1項の業務への従事が次の各号のいずれかに該当する場合には、
   これを許可しないことがある。
   (1) 労務提供上の支障がある場合
   (2) 企業秘密が漏洩する場合
   (3) 会社の名誉や信用を損なう行為や、信頼関係を破壊する行為がある場合
   (4) 競業により、会社の利益を害する場合
```

**原則届出制（推奨）**
```
第○条（副業・兼業）
1. 労働者は、勤務時間外において、他の会社等の業務に従事することができる。
2. 労働者は、前項の業務に従事するにあたっては、事前に会社に所定の届出を
   行うものとする。
3. 第1項の業務に従事することにより、次の各号のいずれかに該当する場合には、
   会社は、これを禁止又は制限することができる。
   (1) 労務提供上の支障がある場合
   (2) 企業秘密が漏洩する場合
   (3) 会社の名誉や信用を損なう行為や、信頼関係を破壊する行為がある場合
   (4) 競業により、会社の利益を害する場合
4. 副業・兼業を行う労働者は、自己の健康管理に十分留意し、本業に支障を
   来さないようにしなければならない。
```

## 退職・解雇

### 退職

**基本パターン**
```
第○条（退職）
労働者が次のいずれかに該当するときは、退職とする。
(1) 退職を願い出て会社が承認したとき、又は退職願を提出して14日を
    経過したとき
(2) 期間を定めて雇用されている場合、その期間が満了したとき
(3) 第○条に定める休職期間が満了し、なお休職事由が消滅しないとき
(4) 死亡したとき
```

**定年制を含む場合**
```
第○条（定年等）
1. 労働者の定年は、満60歳とし、定年に達した日の属する月の末日を
   もって退職とする。
2. 会社は、前項の規定にかかわらず、定年後も引き続き雇用されることを
   希望し、解雇事由又は退職事由に該当しない労働者については、満65歳
   までこれを継続雇用する。

第○条（退職）
労働者が次のいずれかに該当するときは、退職とする。
(1) 退職を願い出て会社が承認したとき、又は退職願を提出して14日を
    経過したとき
(2) 期間を定めて雇用されている場合、その期間が満了したとき
(3) 第○条に定める定年に達したとき
(4) 第○条に定める休職期間が満了し、なお休職事由が消滅しないとき
(5) 死亡したとき
```

### 解雇

**詳細版**
```
第○条（解雇）
1. 労働者が次のいずれかに該当するときは、解雇することがある。
   (1) 勤務状況が著しく不良で、改善の見込みがなく、労働者としての
       職責を果たし得ないと認められるとき
   (2) 勤務成績又は業務能率が著しく不良で、向上の見込みがなく、
       他の職務にも転換できない等就業に適さないと認められるとき
   (3) 業務上の負傷又は疾病による療養の開始後3年を経過しても当該
       負傷又は疾病が治らない場合であって、労働者が傷病補償年金を
       受けているとき又は受けることとなったとき（会社が打切補償を
       支払ったときを含む。）
   (4) 精神又は身体の障害により業務に堪えられないと認められるとき
   (5) 試用期間における作業能力又は勤務態度が著しく不良で、労働者と
       して不適格であると認められるとき
   (6) 事業の運営上又は天災事変その他これに準ずるやむを得ない事由に
       より、事業の継続が困難となったとき
   (7) 事業の運営上又は天災事変その他これに準ずるやむを得ない事由に
       より、事業の縮小・転換又は部門の閉鎖等を行う必要が生じ、
       他の職務への転換が困難なとき
   (8) その他前各号に準ずるやむを得ない事由があるとき
2. 前項の規定により労働者を解雇する場合は、少なくとも30日前に予告を
   するか、又は平均賃金の30日分以上の解雇予告手当を支払う。
   ただし、予告の日数は、平均賃金を支払った日数だけ短縮することができる。
3. 前項の規定は、労働基準法第20条の規定により除外認定を受けた場合、
   又は次の各号のいずれかに該当する労働者については適用しない。
   (1) 日々雇い入れられる者（1ヶ月を超えて引き続き使用されるに
       至った者を除く。）
   (2) 2ヶ月以内の期間を定めて使用される者（所定の期間を超えて
       引き続き使用されるに至った者を除く。）
   (3) 試用期間中の者（14日を超えて引き続き使用されるに至った者を除く。）
```

## 懲戒

### 懲戒の種類

**標準パターン（4段階）**
```
第○条（懲戒の種類）
会社は、労働者が次条のいずれかに該当する場合は、その情状に応じ、
次の区分により懲戒を行う。
(1) けん責
    始末書を提出させて将来を戒める。
(2) 減給
    始末書を提出させて減給する。ただし、減給は1回の額が平均賃金の
    1日分の5割を超えることはなく、総額が一賃金支払期における賃金
    総額の1割を超えることはない。
(3) 出勤停止
    始末書を提出させるほか、○日間を限度として出勤を停止し、
    その間の賃金は支給しない。
(4) 懲戒解雇
    予告期間を設けることなく即時に解雇する。この場合、所轄の労働
    基準監督署長の認定を受けたときは、解雇予告手当を支給しない。
```

### 懲戒事由

**包括的パターン**
```
第○条（懲戒事由）
1. 労働者が次のいずれかに該当するときは、情状に応じ、けん責、減給又は
   出勤停止とする。
   (1) 正当な理由なく無断欠勤が○日以上に及ぶとき
   (2) 正当な理由なくしばしば欠勤、遅刻、早退をしたとき
   (3) 過失により会社に損害を与えたとき
   (4) 素行不良で社内の秩序及び風紀を乱したとき
   (5) 第○条に定める服務規律に違反したとき
   (6) その他この規則に違反し又は前各号に準ずる不都合な行為があったとき
2. 労働者が次のいずれかに該当するときは、懲戒解雇とする。
   ただし、情状により減給又は出勤停止とすることがある。
   (1) 重要な経歴を詐称して雇用されたとき
   (2) 正当な理由なく無断欠勤が○日以上に及び、出勤の督促に応じないとき
   (3) 正当な理由なく無断でしばしば遅刻、早退又は欠勤を繰り返し、
       ○回にわたって注意を受けても改めないとき
   (4) 正当な理由なく、しばしば業務上の指示・命令に従わないとき
   (5) 故意又は重大な過失により会社に重大な損害を与えたとき
   (6) 会社内において刑法その他刑罰法規の各規定に違反する行為を行い、
       その犯罪事実が明らかとなったとき（当該行為が軽微な違反である
       場合を除く。）
   (7) 素行不良で著しく社内の秩序又は風紀を乱したとき
   (8) 数回にわたり懲戒を受けたにもかかわらず、なお、勤務態度等に
       関し、改善の見込みがないとき
   (9) 職責を利用して交際を強要し、又は性的な関係を強要したとき
   (10) パワーハラスメント、セクシュアルハラスメント又はマタニティ
        ハラスメント等のハラスメント行為を行ったとき
   (11) 会社の業務上重要な秘密を外部に漏洩して会社に損害を与え、
        又は業務の正常な運営を阻害したとき
   (12) その他前各号に準ずる程度の不適切な行為があったとき
```

```
第○条（専門業務型裁量労働制）
```

**シフト制の労働時間管理**

```
第○条（勤務シフト）
```

```
(2) 会社が指定する制服・帽子等を着用すること
(3) 爪を短く切り、装飾品を着用しないこと
(4) 作業前後及び用便後は、手指の洗浄・消毒を行うこと

```

## 条文作成時の留意点

### 1. 明確性の原則
- 曖昧な表現を避け、具体的に記載する
- 「等」「その他」の多用を避ける
- 数値は明確に記載する

### 2. 公平性の原則
- 特定の労働者を不当に差別しない
- 合理的な理由のない区別をしない

### 3. 法令遵守の原則
- 労働基準法その他の労働関係法令に違反しない
- 法令の改正に応じて随時見直す

### 4. 実効性の原則
- 実際に運用可能な内容とする
- 企業の実態に即した内容とする

### 5. 労使の信頼関係
- 一方的な押し付けにならないよう配慮
- 労働者の意見を聴取する機会を設ける
