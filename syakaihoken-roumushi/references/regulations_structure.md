## 就業規則の法的根拠

### 労働基準法第89条
常時10人以上の労働者を使用する事業場は、就業規則を作成し、所轄の労働基準監督署長に届け出る義務を負う。

### 「常時10人以上」の判断基準
- **対象**: 正社員、パート、アルバイト等、すべての労働者を含む
- **カウント方法**: 通常の状態で常時使用される労働者数
- **繁閑の差**: 一時的な増減は考慮しない

## 作成・変更プロセス

### 1. 原案作成
- 法的要件の確認
- 企業の実態に合わせた内容の検討
- 業種・規模に応じた特別な配慮事項の検討

### 2. 意見聴取（必須）
**対象**: 労働者の過半数を代表する者
- 過半数労働組合がある場合 → 当該組合
- ない場合 → 民主的方法（投票・挙手等）で選出された労働者代表

**重要**: 
- 使用者は意見を聴く義務があるが、同意を得る義務はない
- ただし、労働者に不利益な変更の場合は、合理性が厳しく判断される

### 3. 届出
**提出書類**:
1. 就業規則（届）
2. 意見書（労働者代表の署名または記名押印があるもの)
3. 就業規則本体

**提出先**: 所轄の労働基準監督署

### 4. 周知（必須）
以下のいずれかの方法で全労働者に周知:
1. 常時各作業場の見やすい場所への掲示または備え付け
2. 書面の交付
3. 磁気テープ、磁気ディスク等に記録し、かつ各作業場に労働者が記録の内容を常時確認できる機器を設置
4. 社内イントラネットへの掲載

**重要**: 周知義務を怠った場合、就業規則は法的に無効と判断される可能性がある。

## 記載事項の三分類

### 絶対的必要記載事項（必ず記載）

#### 1. 労働時間、休憩、休日、休暇

**始業・終業時刻**
```
第〇条（労働時間）
1. 労働時間は、1週間については40時間、1日については8時間とする。
2. 始業・終業の時刻及び休憩時間は、次のとおりとする。
   始業時刻  午前9時00分
   終業時刻  午後6時00分
   休憩時間  正午から午後1時まで
```

**休憩時間**
- 労働時間が6時間を超える場合: 少なくとも45分
- 労働時間が8時間を超える場合: 少なくとも1時間

**休日**
```
第〇条（休日）
1. 休日は、次のとおりとする。
   (1) 土曜日及び日曜日
   (2) 国民の祝日
   (3) 年末年始（12月○日から1月○日まで）
   (4) その他会社が指定する日
```

**年次有給休暇**
```
第〇条（年次有給休暇）
1. 入社後6ヶ月間継続勤務し、全労働日の8割以上出勤した労働者に対して、
   10日の年次有給休暇を付与する。
2. 以後、1年ごとに所定の日数を付与する。
3. 年次有給休暇は、労働者の請求する時季に与える。ただし、事業の正常な
   運営を妨げる場合は、他の時季に変更することがある。
4. 年10日以上の年次有給休暇が付与される労働者については、付与日から
   1年以内に、そのうち5日について、会社が時季を指定して取得させる。
```

#### 2. 賃金

**決定方法**
```
第〇条（賃金）
1. 賃金は、次のとおりとする。
   (1) 基本給  月給制（月額○○円～○○円）
   (2) 諸手当  職務手当、通勤手当、その他
```

**計算・支払方法**
```
第〇条（賃金の支払）
1. 賃金は、毎月末日に締め切り、翌月25日に労働者の指定する銀行口座に
   振り込むことにより支払う。ただし、支払日が休日に当たる場合は、
   その前日に支払う。
2. 賃金は通貨で直接労働者に、その全額を支払う。ただし、法令に別段の
   定めがある場合又は労働者の同意を得た場合は、この限りでない。
```

**昇給**
```
第〇条（昇給）
昇給は、勤務成績等を考慮して、毎年○月に行うことがある。
ただし、会社の業績又は本人の勤務成績等により行わないことがある。
```

**割増賃金**
```
第〇条（時間外労働等の割増賃金）
1. 時間外労働、休日労働及び深夜労働に対する割増賃金は、
   次の割増賃金率により支払う。
   (1) 時間外労働: 25%以上
   (2) 時間外労働（月60時間超）: 50%以上
   (3) 法定休日労働: 35%以上
   (4) 深夜労働（午後10時～午前5時）: 25%以上
2. (1)又は(2)と(4)が重複する場合は、それぞれの割増率を合計した率とする。
3. (3)と(4)が重複する場合は、それぞれの割増率を合計した率とする。
```

#### 3. 退職に関する事項（解雇事由を含む）

**退職**
```
第〇条（退職）
労働者が次のいずれかに該当するときは、退職とする。
(1) 退職を願い出て会社が承認したとき、又は退職願を提出して14日を
    経過したとき
(2) 期間を定めて雇用されている場合、その期間が満了したとき
(3) 第○条に定める休職期間が満了し、なお休職事由が消滅しないとき
(4) 死亡したとき
```

**解雇**
```
第〇条（解雇）
1. 労働者が次のいずれかに該当するときは、解雇することがある。
   (1) 勤務状況が著しく不良で、改善の見込みがなく、労働者としての
       職責を果たし得ないと認められるとき
   (2) 勤務成績又は業務能率が著しく不良で、向上の見込みがなく、
       他の職務にも転換できない等就業に適さないと認められるとき
   (3) 業務上の負傷又は疾病による療養の開始後3年を経過しても当該
       負傷又は疾病が治らない場合であって、労働者が傷病補償年金を
       受けているとき又は受けることとなったとき（会社が打切補償を
       支払ったときを含む。）
   (4) 精神又は身体の障害により業務に堪えられないと認められるとき
   (5) 試用期間における作業能力又は勤務態度が著しく不良で、労働者と
       して不適格であると認められるとき
   (6) 事業の運営上又は天災事変その他これに準ずるやむを得ない事由に
       より、事業の継続が困難となったとき
   (7) 事業の運営上又は天災事変その他これに準ずるやむを得ない事由に
       より、事業の縮小・転換又は部門の閉鎖等を行う必要が生じ、
       他の職務への転換が困難なとき
   (8) その他前各号に準ずるやむを得ない事由があるとき
2. 前項の規定により労働者を解雇する場合は、少なくとも30日前に予告を
   するか、又は平均賃金の30日分以上の解雇予告手当を支払う。
```

### 相対的必要記載事項（制度がある場合に記載）

#### 1. 退職手当
```
第〇条（退職手当）
1. 勤続○年以上の労働者が退職し又は解雇されたときは、この章に定める
   ところにより退職手当を支給する。ただし、自己都合による退職者で、
   勤続○年未満の者には退職手当を支給しない。また、第○条第○項により
   懲戒解雇された者には、退職手当の全部又は一部を支給しないことがある。
2. 退職手当の額は、退職又は解雇時の基本給に勤続年数に応じて定めた
   別表の支給率を乗じて得た額とする。
```

#### 2. 賞与
```
第〇条（賞与）
1. 賞与は、原則として毎年○月及び○月に、在籍する労働者に対し、
   会社の業績等を勘案して支給する。
2. 前項の規定にかかわらず、会社の業績の著しい低下その他やむを得ない
   事由により、支給時期を延期し、又は支給しないことがある。
```

#### 3. 安全衛生
```
第〇条（健康診断）
1. 労働者に対して、採用の際及び毎年1回（深夜労働に従事する者は6ヶ月ごとに
   1回）、定期的に健康診断を行う。
2. 労働者は、前項の健康診断を必ず受けなければならない。
```

#### 4. 表彰・制裁

**表彰**
```
第〇条（表彰）
会社は、労働者が次のいずれかに該当するときは、表彰することがある。
(1) 業務上有益な発明、考案を行い、会社の業績に貢献したとき
(2) 業務能率の改善、経費の節減等に貢献したとき
(3) 災害の防止、その他人命の救助に貢献したとき
(4) その他前各号に準ずる程度の功績があったとき
```

**懲戒の種類**
```
第〇条（懲戒の種類）
会社は、労働者が次条のいずれかに該当する場合は、その情状に応じ、
次の区分により懲戒を行う。
(1) けん責
    始末書を提出させて将来を戒める。
(2) 減給
    始末書を提出させて減給する。ただし、減給は1回の額が平均賃金の
    1日分の5割を超えることはなく、総額が一賃金支払期における賃金
    総額の1割を超えることはない。
(3) 出勤停止
    始末書を提出させるほか、○日間を限度として出勤を停止し、
    その間の賃金は支給しない。
(4) 懲戒解雇
    予告期間を設けることなく即時に解雇する。この場合、所轄の労働
    基準監督署長の認定を受けたときは、解雇予告手当を支給しない。
```

**懲戒事由**
```
第〇条（懲戒事由）
労働者が次のいずれかに該当するときは、情状に応じ、けん責、減給又は
出勤停止とする。
(1) 正当な理由なく無断欠勤が○日以上に及ぶとき
(2) 正当な理由なくしばしば欠勤、遅刻、早退をしたとき
(3) 過失により会社に損害を与えたとき
(4) 素行不良で社内の秩序及び風紀を乱したとき
(5) 第○条に定める服務規律に違反したとき
(6) その他この規則に違反し又は前各号に準ずる不都合な行為があったとき
```

### 任意的記載事項（推奨）

#### 1. 服務規律
```
第〇条（服務規律）
労働者は、次の事項を守らなければならない。
(1) 許可なく職務以外の目的で会社の施設、物品等を使用しないこと
(2) 職務に関連して自己の利益を図り、又は他より不当に金品を借用し、
    若しくは贈与を受ける等不正な行為を行わないこと
(3) 勤務中は職務に専念し、正当な理由なく勤務場所を離れないこと
(4) 会社の名誉や信用を損なう行為をしないこと
(5) 在職中及び退職後においても、業務上知り得た会社、取引先等の
    機密を漏らさないこと
(6) 酒気を帯びて就業しないこと
(7) その他労働者としてふさわしくない行為をしないこと
```

#### 2. ハラスメント防止
```
第〇条（ハラスメントの禁止）
1. すべての労働者は、パワーハラスメント、セクシュアルハラスメント、
   マタニティハラスメント等のハラスメント行為を行ってはならない。
2. 会社は、ハラスメント行為を確認した場合、行為者に対し懲戒処分を
   含む厳正な対処を行う。
3. ハラスメントに関する相談窓口は、○○部とする。
4. 会社は、相談者及び行為者等のプライバシーを保護するとともに、
   相談したこと又は事実関係の確認に協力したこと等を理由として
   不利益な取扱いを行わない。
```

#### 3. 副業・兼業
```
第〇条（副業・兼業）
1. 労働者は、勤務時間外において、他の会社等の業務に従事することが
   できる。
2. 労働者は、前項の業務に従事するにあたっては、事前に会社に所定の
   届出を行うものとする。
3. 第1項の業務に従事することにより、次の各号のいずれかに該当する
   場合には、会社は、これを禁止又は制限することができる。
   (1) 労務提供上の支障がある場合
   (2) 企業秘密が漏洩する場合
   (3) 会社の名誉や信用を損なう行為や、信頼関係を破壊する行為がある場合
   (4) 競業により、会社の利益を害する場合
```

#### 4. 秘密保持・個人情報保護
```
第〇条（秘密保持義務）
1. 労働者は、在職中及び退職後においても、業務上知り得た会社、
   取引先及び従業員等の秘密を他に漏洩してはならない。
2. 労働者は、在職中及び退職後においても、業務上知り得た個人情報を
   他に漏洩し、又は本来の目的以外に使用してはならない。
3. 労働者は、第1項及び第2項に定める書類、データその他の情報を
   記録した媒体等を、退職時に会社に返還しなければならない。
```

## 付属規程

就業規則本体に加えて、以下の付属規程を作成することが推奨される：

1. **賃金規程** - 賃金体系の詳細
2. **退職金規程** - 退職金の計算方法等の詳細
3. **育児・介護休業規程** - 育児・介護休業法に基づく詳細な規定
4. **パートタイマー就業規則** - パートタイム労働者の労働条件
5. **テレワーク規程** - テレワーク実施時のルール
6. **出張旅費規程** - 出張時の旅費の取扱い
