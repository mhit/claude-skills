## 作成・届出手続のチェック

### 基本要件
- [ ] 常時10人以上の労働者を使用している（該当する場合のみ届出義務）
- [ ] 就業規則の適用範囲を明確にしている（正社員のみ/パート含む等）

### 意見聴取手続
- [ ] 労働者の過半数を代表する者から意見を聴取した
- [ ] 意見書に署名または記名押印がある
- [ ] 意見聴取の日付が記載されている

### 届出手続
- [ ] 就業規則（届）を作成した
- [ ] 意見書を添付した
- [ ] 就業規則本体を添付した
- [ ] 所轄の労働基準監督署に届け出た

### 周知措置
- [ ] 以下のいずれかの方法で周知している
  - [ ] 常時各作業場の見やすい場所への掲示または備え付け
  - [ ] 書面の交付
  - [ ] 社内イントラネットへの掲載
  - [ ] その他の適切な方法

## 記載事項のチェック

### 絶対的必要記載事項（必須）

#### 1. 労働時間、休憩、休日、休暇
- [ ] 始業・終業の時刻を明記している
- [ ] 休憩時間を明記している
- [ ] 休日を明記している（週休制/変形休日制等）
- [ ] 年次有給休暇について記載している
- [ ] 年5日の時季指定について記載している（年10日以上付与される労働者対象）

#### 2. 賃金
- [ ] 賃金の決定方法を明記している（基本給・諸手当）
- [ ] 賃金の計算方法を明記している
- [ ] 賃金の支払方法を明記している（銀行振込等）
- [ ] 賃金の締切日・支払日を明記している
- [ ] 昇給に関する事項を記載している
- [ ] 割増賃金の計算方法を記載している

#### 3. 退職に関する事項（解雇事由を含む）
- [ ] 退職事由を明記している（自己都合退職の手続等）
- [ ] 解雇事由を具体的に列挙している
- [ ] 解雇予告に関する事項を記載している

### 相対的必要記載事項（制度がある場合に記載）
- [ ] 退職手当（制度がある場合）
- [ ] 賞与（制度がある場合）
- [ ] 最低賃金額（企業独自の制度がある場合）
- [ ] 食費・作業用品等の負担（労働者負担がある場合）
- [ ] 安全衛生（独自の規定がある場合）
- [ ] 職業訓練（制度がある場合）
- [ ] 災害補償（法定を超える制度がある場合）
- [ ] 表彰及び制裁（制度がある場合）

### 任意的記載事項（推奨）
- [ ] 採用手続
- [ ] 試用期間
- [ ] 人事異動（配置転換、出向、転勤）
- [ ] 服務規律
- [ ] ハラスメント防止
- [ ] 秘密保持義務

- [ ] テレワーク

## 法令遵守のチェック

### 労働基準法遵守

#### 労働時間
- [ ] 法定労働時間（1日8時間、週40時間）を超える場合、36協定を締結している
- [ ] 時間外労働の上限規制を遵守している（月45時間・年360時間）
- [ ] 特別条項の要件を満たしている（年720時間以内、単月100時間未満等）

#### 休憩・休日
- [ ] 労働時間が6時間超の場合、少なくとも45分の休憩を与えている
- [ ] 労働時間が8時間超の場合、少なくとも1時間の休憩を与えている
- [ ] 週1日または4週4日以上の休日を与えている

#### 割増賃金
- [ ] 時間外労働の割増率が25%以上になっている
- [ ] 月60時間超の時間外労働の割増率が50%以上になっている
- [ ] 法定休日労働の割増率が35%以上になっている
- [ ] 深夜労働の割増率が25%以上になっている

#### 年次有給休暇
- [ ] 6ヶ月継続勤務、8割以上出勤の労働者に10日付与している
- [ ] 年10日以上付与される労働者について、年5日の時季指定義務を規定している

#### 解雇
- [ ] 解雇事由が具体的で客観的である
- [ ] 30日前の予告または予告手当の支払いについて記載している
- [ ] 解雇制限（業務上災害、産前産後休業中等）を考慮している

### 労働契約法遵守
- [ ] 無期転換ルールについて考慮している（有期労働契約の場合）
- [ ] 就業規則の不利益変更について合理性を検討している

### 男女雇用機会均等法遵守
- [ ] 性別を理由とする差別的取扱いがない
- [ ] セクシュアルハラスメント防止措置を規定している
- [ ] マタニティハラスメント防止措置を規定している

### 育児・介護休業法遵守
- [ ] 育児休業について規定している（又は別規程で定めている）
- [ ] 介護休業について規定している（又は別規程で定めている）
- [ ] 子の看護休暇について規定している
- [ ] 3歳未満の子を養育する労働者への短時間勤務制度を規定している
- [ ] 3歳以上小学校就学前の子を養育する労働者への柔軟な働き方措置を規定している（2025年4月～）

### 労働施策総合推進法（パワハラ防止法）遵守
- [ ] パワーハラスメント防止措置を規定している
- [ ] 相談窓口を明示している
- [ ] プライバシー保護と不利益取扱い禁止を規定している

### 労働安全衛生法遵守
- [ ] 健康診断の実施について規定している
- [ ] ストレスチェックについて規定している（常時50人以上の事業場）

## 最新法改正対応のチェック

### 2023年4月施行対応
- [ ] 賃金のデジタル払いについて規定している（導入する場合）
- [ ] 月60時間超時間外労働の割増率50%を規定している（中小企業）

### 2024年4月施行対応
- [ ] 労働条件明示事項に「就業場所・業務の変更の範囲」を追加している
- [ ] 有期労働契約の更新上限を明示している（設定している場合）
- [ ] 無期転換申込機会・無期転換後の労働条件を明示している（有期契約の場合）
- [ ] 時間外労働の上限規制を適用している（建設業・自動車運転業務・医師）

### 2025年4月施行対応（予定）
- [ ] 子の看護等休暇の対象拡大（小学校3年生修了まで）を反映している
- [ ] 所定外労働の制限対象拡大（小学校就学前）を反映している
- [ ] 3歳以上小学校就学前の子を養育する労働者への柔軟な働き方措置を規定している
- [ ] 育休取得状況の公表義務を認識している（従業員300人超）

- [ ] 専門業務型裁量労働制を導入している場合、労使協定を締結している

- [ ] 機械設備の点検義務を規定している

### サービス業・小売業
- [ ] シフト制の労働時間管理について規定している
- [ ] 身だしなみに関する規定を設けている（必要な場合）
- [ ] 制服の着用について規定している（必要な場合）

- [ ] 衛生管理に関する特別な規定を設けている
- [ ] 健康状態の申告義務を規定している
- [ ] 制服・帽子等の着用義務を規定している

## 文言・表現のチェック

### 明確性
- [ ] 曖昧な表現（「適宜」「適切に」等）を避けている
- [ ] 具体的な基準・数値を記載している
- [ ] 「等」「その他」の多用を避けている

### 公平性
- [ ] 特定の労働者を不当に差別する内容がない
- [ ] 合理的な理由のない区別がない

### 一貫性
- [ ] 用語の使い方が統一されている
- [ ] 条文間の矛盾がない
- [ ] 他の規程との整合性がとれている

## 最終確認事項

### 形式面
- [ ] 条文番号が正しく付番されている
- [ ] 誤字・脱字がない
- [ ] 日付が正しく記載されている

### 実効性
- [ ] 実際に運用可能な内容である
- [ ] 企業の実態に即した内容である
- [ ] 過度に厳しい又は緩い内容になっていない

### 周知・説明
- [ ] 従業員説明会の実施を計画している
- [ ] Q&A集を作成している
- [ ] 管理職向け研修を計画している

## チェック後のアクション

### すべての項目がOKの場合
1. 労働者代表への意見聴取
2. 労働基準監督署への届出
3. 全従業員への周知
4. 施行日の設定と周知

### 問題点が発見された場合
1. 問題点の洗い出しと整理
2. 修正案の作成
3. 法的リスクの評価
4. 専門家（社会保険労務士等）への相談検討
5. 修正後の再チェック

### 年次見直し
- [ ] 法改正の確認（毎年4月）
- [ ] 運用状況の確認
- [ ] 問題点の洗い出し

### 随時見直し
- [ ] 重要な法改正があった場合
- [ ] 事業内容が大きく変更された場合
- [ ] 従業員数が大幅に増減した場合
- [ ] 労働条件に大きな変更があった場合

## 専門家への相談が推奨されるケース

以下の場合は、社会保険労務士等の専門家への相談を強く推奨します：

- [ ] 初めて就業規則を作成する場合
- [ ] 大幅な改正を行う場合
- [ ] 労働者に不利益な変更を行う場合
- [ ] 特殊な労働時間制度（裁量労働制、変形労働時間制等）を導入する場合
- [ ] 懲戒処分・解雇に関する規定を整備する場合
- [ ] 労使紛争のリスクが高い場合

### 用語集
- **労働者**: 使用者に使用され賃金を支払われる者
- **使用者**: 事業主又は事業の経営担当者その他その事業の労働者に関する事項について事業主のために行為をするすべての者
- **賃金**: 賃金、給料、手当、賞与その他名称の如何を問わず、労働の対償として使用者が労働者に支払うすべてのもの
- **平均賃金**: 算定すべき事由の発生した日以前3ヶ月間にその労働者に支払われた賃金の総額を、その期間の総日数で除した金額
