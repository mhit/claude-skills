## GMOサインの概要

### GMOサインとは
GMOサインは、GMOグローバルサイン株式会社が提供する電子契約サービスです。

### 主な特徴
- **法的効力**: 電子署名法に基づく電子署名により、紙の契約書と同等の法的効力
- **立会人型・当事者型**: 両方の電子署名に対応
- **タイムスタンプ**: 契約日時の証明
- **高セキュリティ**: 暗号化通信、アクセス制御
- **監査証跡**: 誰がいつ何をしたかの記録

### 料金プラン
- **お試しフリープラン**: 月5件まで無料
- **契約印＆実印プラン**: 月額9,680円～
- **エンタープライズプラン**: 大規模組織向けカスタムプラン

## 電子契約の法的有効性

### 法的根拠

#### 1. 電子署名法（電子署名及び認証業務に関する法律）
```
第3条 電磁的記録であって情報を表すために作成されたもの（公務員が職務上
作成したものを除く。）は、当該電磁的記録に記録された情報について本人による
電子署名（これを行うために必要な符号及び物件を適正に管理することにより、
本人だけが行うことができることとなるものに限る。）が行われているときは、
真正に成立したものと推定する。
```

#### 2. e-文書法
多くの文書について、電磁的記録による保存が認められています。

### 労働関係書類の電子化

#### 電子化が認められている主な書類
- 雇用契約書（労働条件通知書）
- 就業規則の周知
- 36協定等の労使協定
- 賃金台帳
- 出勤簿

#### 労働条件通知書の電子交付の要件（労働基準法施行規則第5条）
労働者が希望した場合に限り、以下の方法で電子交付が可能：
1. 電子メールの送信（受信者が出力して書面を作成できるものに限る）
2. FAXの送信
3. SNS等のメッセージ機能

**重要**: 労働者本人の希望が前提

## 導入準備

### ステップ1: アカウント作成

#### 必要事項
- 会社情報（社名、住所、電話番号等）
- 管理者情報
- 支払方法の登録

#### 準備物
- 会社の登記簿謄本（当事者型を利用する場合）
- 法人印鑑証明書（当事者型を利用する場合）

### ステップ2: 電子署名方式の選択

#### 立会人型電子署名
- **特徴**: メールアドレスとアクセスコードで本人確認
- **コスト**: 低コスト
- **用途**: 一般的な契約に適している
- **就業規則・雇用契約**: ✅ 推奨

#### 当事者型電子署名（実印型）
- **特徴**: 電子証明書を使用した厳格な本人確認
- **コスト**: 高コスト（電子証明書の取得が必要）
- **用途**: 高額取引、重要契約
- **就業規則・雇用契約**: △ 通常は不要

**推奨**: 就業規則や雇用契約には立会人型で十分

### ステップ3: テンプレート設定

#### 雇用契約書テンプレートの作成
1. Word/PDFで雇用契約書の雛形を作成
2. GMOサインにアップロード
3. 署名欄の配置設定
4. 必須入力項目の設定

#### 変数機能の活用
- `{{氏名}}`
- `{{入社日}}`
- `{{職種}}`
- `{{基本給}}`

これらを設定しておくと、契約ごとに自動で差し替えられます。

## 就業規則の同意取得

### 法的要件の確認

#### 就業規則の周知義務
労働基準法第106条により、使用者は就業規則を労働者に周知する義務があります。

**周知方法**（電子的方法も可）:
- 常時各作業場の見やすい場所への掲示または備え付け
- 書面の交付
- 社内イントラネットへの掲載等

#### 同意取得の必要性
- **原則**: 就業規則の作成・変更に労働者の同意は不要（意見聴取のみ）
- **例外**: 労働者に不利益な変更の場合、合理性が求められる
- **ベストプラクティス**: 個別の同意を取得しておくことでトラブル防止

### GMOサインでの同意取得手順

#### ステップ1: 就業規則のPDF化
作成した就業規則をPDF形式で保存

#### ステップ2: GMOサインへのアップロード
1. GMOサイン管理画面にログイン
2. 「文書を送信」を選択
3. 就業規則のPDFをアップロード

#### ステップ3: 署名者の設定
1. 署名者として全従業員のメールアドレスを登録
2. 一括送信機能を使用（CSVアップロード可能）

#### ステップ4: 署名欄の配置
```
最終ページに以下を配置：
- 氏名欄
- 署名欄
- 日付欄
```

#### ステップ5: メッセージ設定
```
件名: 【重要】就業規則についてのご確認とご同意のお願い

本文:
○○株式会社
従業員各位

お疲れ様です。人事部です。

この度、就業規則を改定いたしましたので、ご確認ください。

添付の就業規則をご確認いただき、内容にご同意いただける場合は、
電子署名にてご署名をお願いいたします。

ご不明点がございましたら、人事部（xxx@example.com）まで
お気軽にお問い合わせください。

署名期限: 20XX年○月○日

よろしくお願いいたします。
```

#### ステップ6: 送信
- 一括送信を実行
- 従業員にメールが自動送信される

### 従業員側の操作

#### ステップ1: メール受信
GMOサインから「契約書の確認依頼」メールを受信

#### ステップ2: 文書の確認
メール内のリンクをクリックし、就業規則を確認

#### ステップ3: 署名
- 氏名を入力
- 「署名する」ボタンをクリック
- 完了

#### ステップ4: 完了通知
署名完了のメールを受信

## 雇用契約書の締結

### 準備事項

#### 雇用契約書のひな形作成
必須記載事項を含む雇用契約書を作成（Word/PDF）

#### 必須記載事項（労働基準法第15条、同施行規則第5条）
**絶対的明示事項**:
1. 労働契約の期間
2. 就業の場所及び従事すべき業務
3. 始業及び終業の時刻、休憩時間、休日等
4. 賃金の決定、計算及び支払の方法、締切り及び支払の時期
5. 退職に関する事項（解雇の事由を含む）

**2024年4月追加**:
6. 就業場所・業務の変更の範囲
7. 無期転換申込機会（有期契約の場合）
8. 無期転換後の労働条件（有期契約の場合）

### GMOサインでの締結手順

#### ステップ1: 雇用契約書テンプレートの準備
```
雇用契約書

株式会社○○（以下「会社」という。）と{{氏名}}（以下「従業員」という。）は、
次のとおり労働契約を締結する。

第1条（契約期間）
期間の定めなし（又は、20XX年○月○日から20XX年○月○日まで）

第2条（就業の場所及び業務）
1. 就業場所: {{就業場所}}
2. 従事する業務: {{業務内容}}
3. 変更の範囲
   就業場所: {{変更範囲_場所}}
   業務: {{変更範囲_業務}}

（以下、各条項を記載）

20XX年○月○日

会社
住所: ○○
会社名: 株式会社○○
代表者: ○○ ○○  印

従業員
住所: ○○
氏名: {{氏名}}  印
```

#### ステップ2: 新規契約の作成
1. GMOサイン管理画面で「文書を送信」
2. テンプレートを選択
3. 変数に値を入力
   - 氏名: 山田太郎
   - 就業場所: 東京本社
   - 業務内容: 営業業務
   - 変更範囲_場所: 全国の事業所
   - 変更範囲_業務: 営業業務、企画業務

#### ステップ3: 署名順序の設定
```
1. 会社（代表者） - 先に署名
2. 従業員 - 会社署名後に署名
```

#### ステップ4: 送信
会社が先に署名 → 自動的に従業員にメール送信

### 従業員の署名プロセス

1. メール受信
2. リンククリック
3. 契約内容の確認
4. 署名（タップ/クリック）
5. 完了

### 契約完了後

#### 双方への通知
- 会社と従業員の両方に完了メールが送信される
- PDFのダウンロードリンクが含まれる

#### 保管
- GMOサインのクラウド上に自動保存
- 必要に応じてダウンロード・印刷可能
- 法定保存期間中は確実に保管

## 運用のベストプラクティス

### セキュリティ対策

#### アクセス権限の管理
- 管理者権限は必要最小限の人数に限定
- 部門ごとに閲覧権限を設定
- 退職者のアカウントは速やかに削除

#### 二段階認証の有効化
- 管理者アカウントは二段階認証を必須化
- SMS認証またはアプリ認証を推奨

#### 監査ログの定期確認
- 誰がいつどの文書にアクセスしたか定期的に確認
- 不審なアクセスがないか監視

### 運用フロー

#### 新入社員の雇用契約
```
入社2週間前: テンプレートから契約書作成 → 会社署名
入社1週間前: 従業員に送信 → 従業員署名
入社日: 原本の確認・保管
```

#### 就業規則の改定
```
改定決定 → 労働者代表の意見聴取 → 労働基準監督署届出
→ GMOサインで全従業員に送信 → 同意取得 → 施行
```

### トラブルシューティング

#### 従業員がメールを受信していない
**原因**: 迷惑メールフォルダに振り分けられている
**対策**: 
- GMOサインのドメインをホワイトリストに追加
- 別のメールアドレスで再送信

#### 従業員がリンクを開けない
**原因**: アクセスコードの入力ミス、リンクの期限切れ
**対策**:
- 再送信機能を使用
- 電話でサポート

#### 署名を間違えた
**対策**:
- 管理者が文書を取り消し
- 再度送信

### 定期的なメンテナンス

#### 月次
- [ ] 未署名の契約書の催促
- [ ] 完了した契約書の確認
- [ ] ストレージ容量の確認

#### 四半期
- [ ] テンプレートの見直し
- [ ] アクセス権限の見直し
- [ ] 監査ログの確認

#### 年次
- [ ] 法改正に伴うテンプレート更新
- [ ] セキュリティ設定の見直し
- [ ] バックアップの実施

## よくある質問（FAQ）

### Q1: 紙の契約書と同じ効力がありますか？
**A**: はい。電子署名法に基づく電子署名により、紙の契約書と同等の法的効力を持ちます。

### Q2: 従業員が電子契約を拒否した場合は？
**A**: 労働者本人の同意が必要です。拒否する従業員には紙の契約書を用意する必要があります。

### Q3: 印紙税はかかりますか？
**A**: 電子契約の場合、印紙税は課税されません。コスト削減のメリットがあります。

### Q4: データが消失するリスクは？
**A**: GMOサインはクラウド上に安全に保管されており、多重バックアップが行われています。ただし、重要な文書は定期的にダウンロードして自社でもバックアップすることを推奨します。

### Q5: 過去の紙の契約書も電子化すべきですか？
**A**: 法的には義務ではありませんが、管理の効率化のため、可能な範囲で電子化することを推奨します。
