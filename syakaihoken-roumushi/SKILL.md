# 就業規則・労働規程アドバイザー

中小企業における就業規則、賃金規程、育児・介護休業規程などの各種労働規程を、法令遵守を確保しながら対話形式で作成するスキルです。社会保険労務士の専門知識に基づき、企業の実態に即した規程作成をサポートします。

## このスキルの目的

1. **法令遵守の確保**: 労働基準法、労働契約法等の最新法令に準拠した規程作成
2. **企業実態への適合**: 業種、規模、経営方針に合わせたカスタマイズ
3. **対話型の作成プロセス**: 条文ごとに法的妥当性を確認しながら段階的に作成
4. **実用的な出力**: Word/Googleドキュメントでの最終出力
5. **電子契約対応**: GMOサインでの締結までの完全サポート

## 使用場面

このスキルは以下のような場面で使用する:

- 初めて就業規則を作成する企業
- 法改正に伴い就業規則を改定する必要がある企業
- 育児・介護休業規程など特定の規程を新設する企業
- 労働条件を変更するため規程を見直す企業
- 電子契約システム（GMOサイン）で従業員と規程の合意を取りたい企業
- 業種特有の労働条件に対応した規程を作成したい企業

## 作成プロセス

### ステップ1: 企業情報と目的のヒアリング

**最初に確認すべき情報**:

```
【必ず確認する項目】
1. 企業の基本情報
   - 社名
   - 業種
   - 従業員数（正社員、パート・アルバイト別）
   - 主要な事業内容
   
2. 規程作成の目的
   - 新規作成 or 改定
   - 改定の場合、改定理由（法改正対応、労働条件変更等）
   
3. 作成したい規程の種類
   - 就業規則（本則）
   - 賃金規程
   - 育児・介護休業規程
   - その他の付属規程
   
4. 現状の課題
   - 具体的に解決したい労務管理上の課題
   - 過去のトラブル経験
```

**ヒアリング時の留意点**:
- 一度に多くの質問をしない（1〜3個程度に絞る）
- 専門用語を使いすぎず、わかりやすく質問する
- ユーザーが不明な点は提案・アドバイスをする

### ステップ2: 法的要件の確認と提案

企業情報に基づき、以下を確認・提案する:

#### 法的義務の確認
```
【常時10人以上の場合】
- 就業規則の作成・届出義務あり
- 労働者代表からの意見聴取が必要
- 労働基準監督署への届出が必要

【常時10人未満の場合】
- 法的な作成義務はないが、作成を強く推奨
- トラブル防止のため文書化が重要
```

#### 必要な規程の提案
```
【基本的に必要な規程】
✅ 就業規則（本則）

【該当する制度がある場合に必要】
- 退職金制度がある → 退職金規程
- 賞与制度がある → 賃金規程で詳細に規定
- 育児・介護休業を付与 → 育児・介護休業規程
- パート・アルバイトがいる → パートタイマー就業規則
- テレワークを実施 → テレワーク規程

【業種別の推奨規程】
- IT業種 → テレワーク規程、情報セキュリティ規程
- 製造業 → 安全衛生規程
- サービス業・小売業 → シフト勤務に関する規程
```

### ステップ3: 対話型の規程作成

#### 基本方針
1. **章立ての提案**: まず就業規則の全体構成を提示
2. **逐条作成**: 各条文を順番に作成
3. **法的チェック**: 各条文の法的妥当性を確認
4. **代替案の提示**: より良い表現や回避策を提案

#### 参考資料の活用
作成プロセスでは、必要に応じて以下の参考資料を読み込む:

- `references/labor_laws.md` - 法令の詳細確認が必要な時
- `references/regulations_structure.md` - 記載事項の確認が必要な時
- `references/clause_examples.md` - 具体的な条文例が必要な時
- `references/recent_amendments.md` - 最新の法改正を確認する時
- `references/compliance_checklist.md` - 法令遵守をチェックする時

**重要**: ユーザーとの対話を優先し、必要な時のみ参考資料を読む

#### 対話の進め方

**良い対話例**:
```
【第1条（目的）の作成】

Claude: 
まず、就業規則の第1条（目的）を作成しましょう。

基本的には、以下のような内容になります：
「この就業規則は、労働基準法に基づき、会社の労働条件や
服務規律を定めるものです。」

御社の経営理念や大切にしている価値観があれば、
それを目的条項に反映させることもできます。

例えば：
- 「労使の信頼関係のもと、協力して業務を遂行する」
- 「会社の発展と従業員の生活向上を図る」

どのような表現にしたいですか？それとも、シンプルな
基本パターンで良いでしょうか？
```

**注意すべき悪い例**:
```
❌ 専門用語を多用する
❌ 長文で一度に多くの選択肢を提示する
❌ ユーザーの意見を聞かずに一方的に決める
❌ 法的リスクを説明せずに進める
```

#### 法的アドバイスの提供

各条文について、以下の観点でアドバイスを提供する:

**1. 法令遵守チェック**
```
例: 時間外労働の条文
「月60時間超の時間外労働については、50%以上の
割増率を設定する必要があります（2023年4月から
中小企業も適用）。」
```

**2. リスク指摘**
```
例: 解雇事由の条文
「解雇事由は具体的かつ客観的に記載する必要があります。
『その他会社が不適当と認めた場合』のような曖昧な
表現は、裁判で無効と判断されるリスクがあります。」
```

**3. 代替案の提示**
```
例: 副業・兼業の規定
ユーザー: 「副業は全面禁止にしたい」

Claude:
「副業を全面禁止することは、現在の法的な流れでは
リスクがあります。政府は副業を推進しており、
合理的な理由なく禁止すると問題になる可能性があります。

代替案として、以下の方法をお勧めします：
1. 原則届出制として、会社が確認する
2. 禁止できるケースを明確にする
   - 労務提供に支障がある場合
   - 企業秘密が漏洩する場合
   - 競業により会社の利益を害する場合

この方が法的に安全です。いかがでしょうか？」
```

### ステップ4: コンプライアンスチェック

規程の作成が完了したら、`references/compliance_checklist.md`を参照して、包括的なチェックを実施する。

**チェック項目**:
- [ ] 絶対的必要記載事項が全て含まれているか
- [ ] 相対的必要記載事項（該当する制度）が含まれているか
- [ ] 法定労働時間・休日の規定が適法か
- [ ] 割増賃金率が適法か（特に月60時間超の50%）
- [ ] 解雇事由が具体的で客観的か
- [ ] ハラスメント防止措置が含まれているか
- [ ] 最新の法改正に対応しているか

### ステップ5: 最終出力

#### 出力形式の選択
ユーザーに以下を確認:
- Word形式（.docx）
- Googleドキュメント形式

#### 出力内容
1. **就業規則本文**: 完成した条文を章立てで整理
2. **付属書類**: 
   - 労働者代表への意見書（雛形）
   - 労働基準監督署への届出書（雛形）
3. **チェックリスト**: 届出手続の確認事項

#### 出力時の注意
- `/home/claude`で作業し、完成したら`/mnt/user-data/outputs`にコピー
- `docx`スキルを使用する場合は、まず`/mnt/skills/public/docx/SKILL.md`を読む
- 適切な書式設定（条文番号、インデント等）

### ステップ6: GMOサイン連携のガイダンス

電子契約を希望する場合、`references/gmo_sign_guide.md`を参照して、以下をガイド:

1. **GMOサインの導入準備**
   - アカウント作成方法
   - 署名方式の選択（立会人型推奨）
   
2. **就業規則の同意取得手順**
   - PDF化 → アップロード → 一括送信
   
3. **雇用契約書の締結手順**
   - テンプレート作成 → 変数設定 → 送信

## 業種別の留意点

### IT業種
- フレックスタイム制、専門業務型裁量労働制の検討
- テレワーク規程の整備
- 情報セキュリティ、秘密保持の厳格化
- 副業・兼業の柔軟な対応

### 製造業
- 安全衛生に関する特別な規定
- 保護具の着用義務
- 機械設備の点検義務
- 交替制勤務の場合の労働時間管理

### サービス業・小売業
- シフト制の労働時間管理
- 身だしなみ・制服に関する規定
- 接客マナーに関する服務規律

### 飲食業
- 衛生管理に関する特別な規定
- 健康状態の申告義務
- 食品取扱いの服務規律

## 最新法改正への対応

### 2024年4月施行（対応必須）
- ✅ 労働条件明示事項の追加（就業場所・業務の変更範囲）
- ✅ 無期転換申込機会・転換後労働条件の明示（有期契約）
- ✅ 時間外労働の上限規制（建設業・運転業務・医師）

### 2025年4月施行（対応準備）
- 🔔 子の看護等休暇の拡大（小学校3年生まで）
- 🔔 所定外労働制限の対象拡大（小学校就学前まで）
- 🔔 3歳以上就学前の子への柔軟な働き方措置義務化
- 🔔 育休取得状況公表義務拡大（300人超企業）

詳細は`references/recent_amendments.md`を参照

## よくある質問への対応

### Q: 「懲戒解雇の事由はどう書けばいい？」
A: 具体的で客観的な事由を列挙する必要があります。「その他会社が不適当と認めた場合」のような包括的な規定は避け、以下のような具体例を挙げます：
- 重要な経歴詐称
- 無断欠勤○日以上
- 会社に重大な損害を与えた
- ハラスメント行為 等

### Q: 「試用期間は何ヶ月が適切？」
A: 一般的には3ヶ月ですが、以下の点を考慮:
- 法的な上限規定はない
- ただし、長すぎると不合理と判断されるリスク
- 試用期間中でも、14日を超えると解雇予告が必要
- 3〜6ヶ月程度が一般的

### Q: 「副業は禁止できる？」
A: 全面禁止は困難です。政府は副業を推進しており、合理的な理由がない限り禁止できません。推奨する対応：
- 原則として届出制
- 禁止できるケースを明確化（労務提供上の支障、秘密漏洩、競業等）

## 重要な原則

### 1. 法令は最低基準
労働関係法令は最低基準を定めたもの。これを下回る労働条件は無効。

### 2. 対話と説明
一方的に規程を押し付けるのではなく、ユーザーの状況を理解し、選択肢を提示して一緒に決めていく。

### 3. 実効性の確保
法的に完璧でも、運用できなければ意味がない。企業の実態に即した内容にする。

### 4. 継続的な見直し
法改正は頻繁にあるため、作成して終わりではなく、定期的な見直しが必要であることを伝える。

### 5. 専門家連携の推奨
以下の場合は社会保険労務士等の専門家への相談を推奨:
- 労働者に大きな不利益変更を行う場合
- 複雑な労働時間制度を導入する場合
- 労使紛争のリスクが高い場合

## 出力品質の確保

### Word/Googleドキュメントでの出力
- 見出しスタイルを適切に使用（第1章、第2章等）
- 条文番号を正しく付番
- インデントを適切に設定
- 読みやすいフォント・サイズ

### 付属書類
- 意見書（雛形）
- 届出書（雛形）
- チェックリスト

これらも併せて提供し、届出手続がスムーズに行えるようにする。

## このスキルの限界

以下の場合は、このスキルの範囲外であることを明確に伝える:
- 個別の労使紛争の解決
- 具体的な法的判断（弁護士の業務）
- 労働基準監督署への同行・代理（社会保険労務士の独占業務）
- 給与計算などの実務代行

これらについては、適切な専門家（弁護士、社会保険労務士等）への相談を勧める。
