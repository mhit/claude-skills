# 神殺データベース

## 概要
神殺（しんさつ）は、四柱推命において命式に現れる特殊な星で、吉神と凶神に分類されます。吉神は幸運や才能を、凶神は注意すべき点や試練を示します。

---

## 吉神（きっしん）

### 天乙貴人（てんおつきじん）
**判定方法**:
日干または年干から判定。以下の対応表による。

| 日干/年干 | 天乙貴人の地支 |
|----------|--------------|
| 甲・戊 | 丑・未 |
| 乙・己 | 子・申 |
| 丙・丁 | 亥・酉 |
| 庚・辛 | 寅・午 |
| 壬・癸 | 巳・卯 |

**意味**:
- 最高位の吉神、貴人の守護
- 困難に遭っても援助者が現れる
- 社会的地位や名声を得やすい
- 品位があり、人より尊敬を受ける
- 危機一髪の時、死中に活を得る

**解釈文**:
```
例え、親の遺産なくとも、生来、自己に備わる福運が強く、中晩年に必ず発達する。
又、生れの如何に関わらず、財運が有り、困難に頻しても援助者が現れ、救いの手を差し伸べてくれる。
一生大手術を受ける事無く、悪疫、伝染病等に倒れる事はない。
又、悪死、急死、頓死からも免れて、危機一髪の時、死中に活を得る。
正しいプライドと、身に品位強く、卑しさ無く、目上の引立てを受け発達し、一生の幸せが約束される。
```

**特殊な組み合わせ**:
- 天乙貴人 + 印綬（同柱）: 高位・高官に登る
- 天乙貴人 + 帝旺: 人を引き付ける魅力が強く、頭領やプロジェクトリーダーに適す

## 凶神（きょうしん）

### 白虎（びゃっこ）/ 災煞
**判定方法**（3つの流派あり、実装時は選択可能に）:

**方法1: 三合局起点（推奨・多数派）**
- 申子辰 → 午を白虎とする
- 亥卯未 → 酉を白虎とする
- 寅午戌 → 子を白虎とする
- 巳酉丑 → 卯を白虎とする

**方法2: 日柱干支による判定**
- 甲辰、乙未、丙戌、丁丑、戊辰、壬戌、癸丑の日柱を持つ人

**方法3: 日支からの判定**
| 日支 | 白虎の地支 |
|-----|----------|
| 子 | 申 |
| 丑 | 未 |
| 寅 | 午 |
| 卯 | 巳 |
| 辰 | 辰 |
| 巳 | 卯 |
| 午 | 寅 |
| 未 | 丑 |
| 申 | 子 |
| 酉 | 亥 |
| 戌 | 戌 |
| 亥 | 酉 |

**意味（両面性あり）**:

**吉作用**:
- 虎のように強くて勇猛なエネルギー
- 強力な推進力、決断力
- 専門家としての気運
- 女性が白虎殺と魁罡殺を同時に持つと社会的に大きな人物になる
- 活人業（医師、裁判官、軍人、警察、宗教人、占術家）で凶意を吉転できる

**凶作用**:
- 血光の災い、怪我、手術
- 病気・疾病・傷害
- 交通事故・水の事故
- 思いがけない災い
- 独断的で独善的になる傾向

**解釈文**:
```
遺伝的に胃腸弱く、生涯、怪我、手術等に悩まされる剣難の相である。
【甲】【乙日】生れの命にこの星あれば：胃潰瘍、胃癌にかかる率が特に高い。
病気の暗示の有る星で、幸運が少なく、不運が続く。
【白虎二つ】水難・交通事故・横死等不測の災厄に注意。

しかし、虎のような強い推進力を持ち、医療・法律・軍事・宗教などの
「活人業」に転じることで、この凶意を大きな成功のエネルギーに転換できる。
```

**対策・活用法**:
- 医師、看護師、救命士などの医療職
- 裁判官、弁護士などの法律職
- 警察官、自衛官、消防士
- 宗教家、占術家、カウンセラー
- これらの「命を救う職業」で白虎のエネルギーを吉転可能

### 血刃（けつじん）
**判定方法**（固定表: 年支または日支基準）:

| 年支/日支 | 血刃 | 年支/日支 | 血刃 |
|---------|-----|---------|-----|
| 子 | 戌 | 午 | 辰 |
| 丑 | 酉 | 未 | 卯 |
| 寅 | 申 | 申 | 寅 |
| 卯 | 未 | 酉 | 丑 |
| 辰 | 午 | 戌 | 子 |
| 巳 | 巳（同支） | 亥 | 亥（同支） |

**月支→血刃支**の対応表:
- 子→午、丑→未、寅→申、卯→酉、辰→戌、巳→亥
- 午→子、未→丑、申→寅、酉→卯、戌→辰、亥→巳

**意味（両面性あり）**:

**吉作用**:
- 野心家で成功のために血を流すことも厭わない
- 幸運を自分からつかみ取る
- 起業家精神、開拓精神
- 困難を乗り越える力

**凶作用**:
- 怪我、出血、手術
- 血液に関する病気
- 口舌の災い
- 女性の場合、お産が重くなる

**解釈文**:
```
怪我・外科手術の暗示がある。特に交通事故や刃物での負傷に注意。
しかし、この星を持つ人は野心が強く、困難を恐れず前進する力を持つ。

現代では力仕事が減少しているため、血刃の凶作用は以前より軽減されている。
料理や運転時には特に注意が必要だが、定期的な健康診断を受けることで
リスクを管理できる。

医療・外科・救命分野に進むことで、この星のエネルギーを吉転できる。
```

**柱ごとの意味**:
- **年柱**: 家系や先祖からの影響、幼少期の怪我
- **月柱**: 職場や家庭内での怪我・病気、30代前後
- **日柱**: 配偶者やパートナーの健康問題、40代
- **時柱**: 子供や晩年の健康問題、50代以降

**相互作用**:
- **天乙貴人あり**: 災厄が軽減される（凶作用×0.5）
- **白虎と同時**: 注意が必要（凶作用×1.5）
- **複数の柱にある**: より慎重に（凶作用累積）
- **亡神と合わさる**: 「戦略的計画」と「怪我か外科手術の暗示」の両面
- **医薬・天医と重なる**: 「刃→医療」の形で吉転（医療関係者に最適）

**対策と活用法**:
- 医師、外科医、救命士、看護師などの医療職
- 安全運転、刃物の取り扱いに注意
- 定期健康診断（特に血液検査）
- スポーツや武道で安全にエネルギー発散
- 起業家、開拓者としての資質を活かす

## 神殺の優先順位

複数の神殺が命式に現れる場合の解釈順序:

1. **天乙貴人**（最優先の吉神）
2. **羊刃・白虎**（重大な凶神）
3. **天徳貴人・月徳貴人**（強力な吉神）
4. **孤辰・寡宿**（人生への影響大）
5. **駅馬**（生活様式への影響）
6. **その他の神殺**

## 神殺の相互作用

### 吉神 + 吉神
- 効果が相乗的に強まる
- 特に天乙貴人 + 天徳貴人は最強

### 吉神 + 凶神
- 凶神の作用が緩和される
- 完全には打ち消されないが、災いが軽減

### 凶神 + 凶神
- 注意が必要
- 特に白虎 + 羊刃は重大
