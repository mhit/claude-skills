# 四柱推命 実践ガイド

## 命式作成の手順

### ステップ1: 四柱の算出

```
1. 万年暦から年月日時の干支を取得
   - 年柱: 生まれ年の立春から始まる干支
   - 月柱: 節入り(節気の開始)からの干支
   - 日柱: 生まれ日の干支
   - 時柱: 生まれ時刻の干支(2時間ごと)

2. 節入りの確認
   - 生まれ日が節入り前か後かで月柱が変わる
   - 立春前の生まれは前年扱い

3. 時柱の調整
   - 時刻不明の場合は正午(12:00)を使用
   - その旨を明記
```

### ステップ2: 蔵干の算出

```
1. 月柱の地支から蔵干を算出
   - 節入りからの日数をカウント
   - 余気→中気→本気の順に該当判定

2. 月支元命の特定
   - 算出された蔵干と日干の関係から通変星を導出
   - これが最重要な「中心星」となる
```

### ステップ3: 通変星の算出

```
日干を基準として、他の七つの干との関係を計算:

1. 年柱天干 vs 日干 → 年柱天干の通変星
2. 月柱天干 vs 日干 → 月柱天干の通変星
3. 時柱天干 vs 日干 → 時柱天干の通変星
4. 各地支の蔵干 vs 日干 → 地支の通変星

計算式:
- 五行関係を特定(比・生・剋・被剋・被生)
- 陰陽関係を特定(同性・異性)
- マトリクスから通変星を決定
```

### ステップ4: 十二運の算出

```
日干が各地支でどのエネルギー状態にあるかを判定:
- 日干と各地支の組み合わせで決定
- 例: 甲(日干) + 寅(地支) = 建禄
```

### ステップ5: 日干の強弱判断

```
以下の4要素で総合判断:

1. 月令チェック
   - 春生まれの木、夏生まれの火など → 得令(強)
   - 反対の季節 → 失令(弱)

2. 通根チェック
   - 地支の蔵干に日干と同じ五行が何個あるか
   - 多いほど強い

3. 透干チェック
   - 他の天干に日干と同じ五行が何個あるか
   - 多いほど強い

4. 生助チェック
   - 日干を生じる五行(印星)が多いか

結論:
- 上記が多い → 身強 → 抑制が必要(食傷・財・官殺が喜神)
- 上記が少ない → 身弱 → 支援が必要(印・比劫が喜神)
```

### ステップ6: 格局判定

```
1. 内格か外格かを判定
   - 五行がバランスしている → 内格
   - 五行が極端に偏っている → 外格

2. 具体的な格局を特定
   - 内格: 月支元命の通変星で判定
   - 外格: 最も強い五行で判定(従旺、従児、従財、従殺)
```

### ステップ7: 大運の算出

```
1. 立運年齢の計算
   - 陽年男性・陰年女性: (次の節入り - 生日) ÷ 3 (順行)
   - 陰年男性・陽年女性: (生日 - 前の節入り) ÷ 3 (逆行)

2. 10年ごとの大運干支をリストアップ
   - 順行: 月柱の次の干支から順番に
   - 逆行: 月柱の前の干支へ逆順に

3. 各大運の喜忌判定
   - 大運の五行が喜神 → 良好な10年
   - 大運の五行が忌神 → 試練の10年
```

## 簡易フローチャート

```
ユーザー入力
    ↓
万年暦で四柱取得
    ↓
蔵干算出
    ↓
通変星・十二運算出
    ↓
日干強弱判断
    ↓
喜神・忌神決定
    ↓
格局判定
    ↓
大運算出
    ↓
統合解釈
    ↓
レポート作成
```

## 重要な注意点

1. **節入りの確認**: 月の変わり目は要注意
2. **時刻の精度**: 不明な場合は正午を使用、その旨を明記
3. **複数の要素の総合判断**: 単一の星だけで判断しない
4. **文化的配慮**: 凶運も「成長の機会」として表現
