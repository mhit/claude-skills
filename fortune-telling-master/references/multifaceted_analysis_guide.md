# 多面的分析ガイドライン

## 基本原則

四柱推命の専門的な鑑定では、**意図的に矛盾する表現を含める**ことが重要です。これは命式の複雑さを正確に反映するためです。

### なぜ矛盾を含めるのか

1. **命式は多面的**: 一人の人間は単純な一面だけではない
2. **状況依存性**: 同じ星でも環境により異なる表現となる
3. **陰陽のバランス**: 対立する要素の共存が自然
4. **成長の余地**: 矛盾は発展の可能性を示す

### 矛盾の例（実際の鑑定書から）

```
○ 良い例:
「物事を実直正確に拘わりを持って、信用と真面目さを基調に処理する人。」
    ↓ しかし同時に
「複数の物事を同時に行う時や、指示命令が的確でない場合はパニックに陥り、
 事の責任を追求されると、他力本願な言い訳や他人に頼る傾向も持つ。」

→ この矛盾は、環境が整っているときと混乱しているときで
   全く異なる面が出ることを示している
```

```
○ 良い例:
「常に物事を仕切り、独立心が強く、自分で自分を律し、
 事の責任をしっかり負う人。」
    ↓ でも
「人の世話焼きや、奉仕、貢献といったお節介性が非常に強い人で、
 常に自分のことより、周囲や人の為に事を行なうことが好き。」

→ リーダーシップ(比肩)と奉仕精神(印綬)の両面を持つことを示す
```

## 多面的分析の実践

### ステップ1: 各星の影響を個別に記述

命式に現れる各通変星、十二運、神殺の影響を**それぞれ独立して**記述します。

例:
- 月支元命が傷官 → 専門技術・職人気質の記述
- 食神が命式にある → 自由・楽天的な性格の記述
- 印綬が命式にある → 奉仕・世話焼きの記述

### ステップ2: 矛盾を恐れない

上記の記述が矛盾しても、**そのまま併記**します。

プロの鑑定書では冒頭に必ずこう記載します:
```
「本判定では、より確実なものとするために、さまざまな視点から判定を
行なっており、そのため判定文の中に矛盾する表現が現れる場合があります。
例）穏やかな性格と記されている行と、激しい性格と記されている行が存在
するなど。こういう場合は、互いを打ち消しあっていますので、より強い方
の特性を弱めて解釈して下さい。」
```

### ステップ3: 文章構成のパターン

#### パターン1: 段落ごとに異なる側面

```markdown
第1段落: 職業適性（傷官の影響）
　職業的には、専門知識、技術、制作、職人、技芸芸術、
　マニアック、オタク的な職業適性を持つ。
　物事を実直正確に拘わりを持って処理する人。

第2段落: リーダーシップ（比肩の影響）
　自営、経営者、経営代行、団体幹部、プロジェクトリーダー。
　常に物事を仕切り、独立心が強く、自分で自分を律する人。

第3段落: 奉仕精神（印綬の影響）
　生来、人の世話焼きや、奉仕、貢献といったお節介性が
　非常に強い人で、常に自分のことより周囲の為に事を行なう。

第4段落: 自由主義（食神の影響）
　社会的な地位、名誉、権威を得る為の重く堅苦しい人生を
　生きるより、完璧な自由主義を貫き、明るく楽しく過ごす。
```

#### パターン2: 同一段落内での対比

```markdown
「明敏活発、強引で、目下の者に威張りたがる傾向も持つが、
 どんな場合でもリーダーシップを取り、決断力があるが、
 自分が処理実行の当事者の場合、用心深く用意周到である。」

→ 前半と後半で矛盾する特性を一文で表現
```

### ステップ4: 条件付き記述

矛盾を解消するために「～の場合」という条件を付ける:

```markdown
「上司や頭領から任務指示、命令を具体的にしっかり受け、
 物事を忠実に実直に行うことは得意な人。
 【しかし】複数の物事を同時に行う時や、指示命令が的確で
 ない場合はパニックに陥る。」

→ 条件を明示することで、矛盾ではなく状況依存性を示す
```

## 各セクションでの記述量

### 1. 職業適性: 3-5段落
- 専門性（傷官・印綬系）
- リーダーシップ（比肩・劫財系）
- 社交性（食神・正財系）
- 管理能力（正官・偏官系）

### 2. 性格: 5-8段落
- 基本的な気質
- 対人関係
- 感情の傾向
- 思考パターン
- 行動様式

### 3. 愛情・結婚: 2-3段落
- 恋愛傾向
- 理想のパートナー像
- 家庭生活の在り方

### 4. 財運: 2-3段落
- 金銭感覚
- 蓄財傾向
- 投資・運用スタイル

### 5. 健康: 1-2段落
- 体質的傾向
- かかりやすい病気
- 注意すべき点

### 6. 人生の流れ: 2-3段落
- 重要な年齢期間
- 大運の影響
- 生涯の傾向

## 使用する表現のトーン

### 良い例:
- 「～する人である」（断定的）
- 「～の傾向が強い」（やや断定的）
- 「～することが多い」（一般的傾向）

### 避けるべき表現:
- 「～かもしれません」（弱すぎ）
- 「～の可能性があります」（曖昧すぎ）

### 注意喚起:
- 「注意を！」（強い警告）
- 「注意」「注意点」（標準的警告）

## 神殺の扱い

吉神・凶神は**必ず記述**します:

### 吉神の例:
```
「例え、親の遺産なくとも、生来、自己に備わる福運が強く、
 中晩年に必ず発達する。又、生れの如何に関わらず、財運が有り、
 困難に頻しても援助者が現れ、救いの手を差し伸べてくれる。」
```

### 凶神の例:
```
「遺伝的に胃腸弱く、生涯、怪我、手術等に悩まされる剣難の相である。
 病気の暗示の有る星で、幸運が少なく、不運が続く。」
```

## 実践チェックリスト

鑑定書を作成する際、以下を確認:

- [ ] 矛盾する記述が3箇所以上ある
- [ ] 各通変星の影響が個別に記述されている
- [ ] 職業適性が多面的（3種類以上）
- [ ] 性格が単一ではなく複雑に描写されている
- [ ] 神殺（吉神・凶神）が記述されている
- [ ] 具体的な年齢・期間が示されている
- [ ] 注意点・警告が含まれている
- [ ] 断定的なトーンを維持している

これらを守ることで、プロフェッショナルな鑑定書が完成します。
