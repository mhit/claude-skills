## 解釈文の構造分析

### 特徴1: 多面的記述（意図的な矛盾を含む）

冒頭に以下の注意書きあり:
```
「判定は、命式表に現れる顕著な特性のうち、より確実なものを示したものです。
本判定では、より確実なものとするために、さまざまな視点から判定を行なっており、
そのため判定文の中に矛盾する表現が現れる場合があります。
例）穏やかな性格と記されている行と、激しい性格と記されている行が存在するなど。
こういう場合は、互いを打ち消しあっていますので、より強い方の特性を弱めて解釈して下さい。」
```

**重要**: この多面的記述こそがプロフェッショナルな鑑定の証

上司や頭領から任務指示、命令を具体的にしっかり受け、物事を忠実に実直に行うことは得意な人。

→ 環境が整っている時とそうでない時で全く異なる面が出る

## 奉仕精神の記述パターン（印綬の影響）

```
生来、人の世話焼きや、奉仕、貢献といったお節介性が非常に強い人で、
常に自分のことより、周囲や人の為に事を行なうことが好きで、
基本的な個性としている人である。

対人や周囲に行なった事を、喜ばれ、感謝されれば、殊更自分の励みや喜びに反映する人で、
多くは社会貢献や、奉仕、ボランティアなどの仕事につく人も多い。
本業が別にあっても、プライベートでこの種の活動を積極的にする人の多い。
```

**重要な条件**:
```
この人に大切な事は、世話の焼き甲斐や、奉仕のし甲斐、
して上げて良かったと思える人、感謝や評価をしてくれる人を選ぶことである。

それが無い場合、行なった自分が報われず、悔しく、辛く、悲しく、惨めで、
メゲやイライラを起こし、人間不審に陥り、人に対する思いやり、
周囲や社会に対する奉事の心を失うので注意。
```

**本質的な矛盾**:
```
本来、奉仕や貢献は、見返りを請わないものであるが、
この人を上手く活かすには、この人の上記の行動を素直に喜び、感謝し、賛美すれば、
その持ち味、魅力がしっかり出ることとなる。
```

**実践的注意**:
```
この人の注意する点は、対人で、困ったふりをする人や、
この人の性質を利用して騙す人が現代社会には多いので、人を十分選び、
この人の奉仕、世話焼きが、対人の人生の発展に役立つか否かをしっかり考える必要がある。
```

## 自由主義の記述パターン（食神の影響）

```
社会的な地位、名誉、権威、権力を得る為の、重く堅苦しい人生を生きるより、
完璧な自由主義を貫き、常に明るく楽しく、夢、希望を追い求め、
自分の趣味、レジャーを、人生の目標や活力や原動力にする人で、
美、自由、快楽、若さ、活気、陽気といった総体像の人である。

枠組みや立法の強い国家、社会、家庭では、極めて生きにくい人である。

この人は、自由の見地で、仕事や社会や家庭をつくることがベストで、
自分が環境によって制御されると、恐怖、臆病、迷い、不安が出、
社会的に無機質となり、極楽トンボとなる。
```

**価値観**:
```
社会的な地位欲、名誉欲はほとんど持っていない人である。
物事は、深く見るより表面にすぐ見えるものを好み、レジャーやリラックスやお祭が大好きであり、喜ぶ。
良い遊びや、自由、レジャーを強化すれば、その為の稼ぎや、仕事、義務はしっかり行うが、
それらが実現出来ない環境であると何もしない。
この人には、地位、名誉、世間体より、自由、楽しみ、趣味、気楽の世界である。

**対人関係**:
```
他人の長所、美点を見抜く事が早く、それを素直に誉めて、決して人をけなす様な、嫌な事は言わない。
又、けなされ、怒られる事を嫌い、極度に臆病で猜疑心も強い。

一見、過去に拘わらず、サラリとしているので、人には好かれ、社交的で友人は多いように見えるが、
この人から積極的に人の輪や、人的交際に入って行く事は極めて少なく、嫌いで、
常に自分の仲間内で遊んでいる人である。

**戦略性**:
```
権謀術策に弱く、戦略的な行為・行動は下手だし嫌いで、

**外見的特徴**:

## 性格の複合的記述パターン

### 基本性格（複数の星の統合）

```
物事に忠実で拘わり強く、実直で正義感が強く、常識家で、情熱に富み、活動的である。
一面、防衛本能が強く、自己中心的傾向が強く、他力本願が強い。

明敏活発、強引で、目下の者に威張りたがる傾向も持つが、
どんな場合でもリーダーシップを取り、決断力があるが、
自分が処理実行の当事者の場合、用心深く用意周到である。
約束を軽んじる傾向と、他力本願な傾向を併せ持つ。

明るく大らかで、発展性と向上心に富み、愛嬌や社交性もあり、社会的に好かれるが、
決断・実行に迷いが多く、事の処理に横着な傾向を持つ。
```

### 短文での総括

```
人当たりよく自在な社交性を持ち、専門知識・専門技芸で活躍する人で、物事に熱心で清潔好き。
短気で衝動的、警戒本能強いが大物。
気分屋で無計画、勘に頼る性質が強く、金銭にこまかい人。
```

## 偏官（七殺）の詳細解釈

```
明敏で頑張り屋で、派手好み、感情のむら気が強く、怒りやすく、短気で、
冷静さを欠くので、物事の失敗が多い。しかし、先を見通す力を持っているので、
その時々に応じて、素早く臨機応変に事を処置する才能がある。

自己中心性の強さや、人を見下し、人の意見を聞き入れない事が多いので注意、
人への思いやりを忘れなければ良い。

一度苦境に陥ると、自分を助けてくれる知人、友人、部下が無くて、
浮かび上がれなくなる事があるので、普段より人の為に尽くす事が必要である。
```

### 健康（偏官）

```
体質（病気）的には、丈夫だが、我ままなので、人の意見を入れず不養成になりがち。
かかりやすい病気は、心臓病・眼疾・不眠症・喘息・咽喉カタル・歯痛・火傷等。
```

### 愛情（偏官）

```
愛情（性情）的には、美しさに対する関心が強いので面食いで、美男美女を求めることが多く、
交際が深くなると、人情や同情、愛情につまずくケース多い。

セックスは精力的ではないので、自己中心に成りやすく、
相手を喜ばせるより、自分の為のテクニックが中心となる。
```

### 財運（偏官）

```
財運的には、財より名誉の方を重んじるタイプで、年収がいくらあるかなどと、
他人に誇るのも一種の自己顕示欲から来る表現で、身を飾ったり、骨董品に凝ったりして、
晩年になっても蓄えが少ない傾向に注意。
```

### 人生の傾向（偏官）

```
幼少にして、両親の元を離れて、他家に養われる事が多く、
人の上に立ちたいという希望が強く、人を押し退けても進もうとするバイタリティーが強い。

平生、人や部下の為に尽くして、思いやりを持った行動をし、人の言を聞く事が大切である。

３４才〜５５才が人生のハイライトである。

生涯、部下に適当な人が見付からず、裏側の事が表面化する事があり、
思わぬ失敗をする事もあるので注意。
```

## 吉神の影響（天乙貴人など）

```
例え、親の遺産なくとも、生来、自己に備わる福運が強く、中晩年に必ず発達する。
又、生れの如何に関わらず、財運が有り、困難に頻しても援助者が現れ、救いの手を差し伸べてくれる。

一生大手術を受ける事無く、悪疫、伝染病等に倒れる事はない。
又、悪死、急死、頓死からも免れて、危機一髪の時、死中に活を得る。

技芸の素質に恵まれ、品位があり、人より尊敬を受け、生活豊かで、
色彩感覚や美術、芸術の素養有るが、孤独の暗示有り。

人を引き付ける魅力が強く、機敏で、先見の明強く、頭領やプロジェクトリーダーに適す。

多数の従業員を使用し、人の頭となり、社会的名声を得て大活躍する。

正しいプライドと、身に品位強く、卑しさ無く、目上の引立てを受け発達し、一生の幸せが約束される。

決断力が優れ、退く事で「利」を得る適性で、物事の無理押し、ごり押しは極めて危険である。

聡明で敏捷、目先が早い。女子は多産の人が多い。
