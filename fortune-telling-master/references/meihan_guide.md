# 姓名判断 実践ガイド

## 五格算出の手順

### ステップ1: 流派の確認

```
ユーザーに確認:
「熊崎式(伝統的)と現代式、どちらをご希望ですか?」

- 未指定の場合は熊崎式を推奨
- 流派の違いを簡潔に説明
```

### ステップ2: 各文字の画数算出

```
1. 姓の各文字の画数を取得
2. 名の各文字の画数を取得
3. 特殊部首のチェック(熊崎式の場合)
   - 氵(さんずい) → 4画
   - 艹(くさかんむり) → 6画
   - など(data_tables.md参照)

4. 旧字体・新字体の確認
   - 沢 vs 澤
   - 広 vs 廣
   - など
```

### ステップ3: 五格の計算

```
姓名構造を判定:
- 姓2字×名2字 → 最も一般的
- 姓1字×名1字 → 霊数適用
- 姓1字×名2字 → 霊数適用
- 姓2字×名1字 → 霊数適用
- 姓3字×名2字 → レアケース

構造に応じた計算式を適用(meihan_theory.md参照)
```

### ステップ4: 各格の吉凶判定

```
1. 各格の画数を1~81の範囲に正規化
   - 82以上 → 81で割った余り
   - 例: 82 = 2, 83 = 3

2. data_tables.mdから画数の意味を参照
   - 吉凶等級
   - キーワード
   - 詳細な解釈

3. 格の重要度を考慮
   - 人格 > 総格 > 地格 > 外格 > 天格
```

### ステップ5: 陰陽配列の分析

```
各文字の画数が:
- 奇数 → 陽(○)
- 偶数 → 陰(●)

パターンを評価:
- ○●○● または ●○●○ → 理想的
- ○○ ○○ → 全陽(不吉)
- ●● ●● → 全陰(不吉)
- ○●○ または ●○● → 挟み(不吉)
- ○○●○○ → 中折れ(不吉)
```

### ステップ6: 三才配置の分析

```
1. 天格・人格・地格の下一桁を取得
2. 各数字を五行に変換
   - 1,2 → 木
   - 3,4 → 火
   - 5,6 → 土
   - 7,8 → 金
   - 9,0 → 水

3. 天→人、人→地の関係性を評価
   - 相生 → 吉
   - 相剋 → 凶
   - 比和 → 中

4. data_tables.mdの三才配置マトリクスを参照
```

## 簡易フローチャート

```
ユーザー入力(氏名+流派)
    ↓
各文字の画数算出
    ↓
五格計算
    ↓
各格の吉凶判定
    ↓
陰陽配列分析
    ↓
三才配置分析
    ↓
総合評価
    ↓
レポート作成
```

## 改名提案の手順

```
現在の名前に問題がある場合:

1. 改善すべき格を特定
   - 凶数がある格
   - 三才配置の相剋部分

2. 姓は変更できないため、名の画数を調整
   
3. 目標画数の設定
   - 人格を吉数に(特に31, 32, 24, 15)
   - 総格を吉数に
   - 三才配置を相生に

4. 逆算で名の画数を計算
   - 人格 = 姓の最後 + 名の最初
   - 名の最初 = 目標人格 - 姓の最後
   - 名の合計 = 目標地格
   - 名の最後 = 名の合計 - 名の最初

5. 該当する画数の漢字を検索
   - 意味の良い漢字を選択
   - 読みやすさ、書きやすさも考慮

6. 複数候補を提示
```

## 重要な注意点

1. **流派の一貫性**: 一度選んだ流派で最後まで統一
2. **画数の正確性**: 特に熊崎式の特殊ルールに注意
3. **霊数の適用**: 一文字名の場合は必ず1画加算
4. **総合判断**: 一つの凶数だけで全否定しない
5. **改名の現実性**: 読みやすさ、覚えやすさも重要
