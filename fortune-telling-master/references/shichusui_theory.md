# 四柱推命 理論体系

## 基礎概念

### 十干(天干)
宇宙の純粋なエネルギーを表す10の記号:
- 陽干: 甲(木)、丙(火)、戊(土)、庚(金)、壬(水)
- 陰干: 乙(木)、丁(火)、己(土)、辛(金)、癸(水)

### 十二支(地支)
地上の複合的なエネルギーを表す12の記号:
- 子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥
- 各支は内部に複数の天干(蔵干)を含む

### 陰陽五行説
**五行**: 木、火、土、金、水
**相生**: 木→火→土→金→水→木(支援・育成)
**相剋**: 木→土、土→水、水→火、火→金、金→木(制御・活用)

## 通変星体系

日干を基準とした他の干との関係性を10種類で表す:

### 自我の星
- **比肩**: 同五行・同陰陽 → 独立性、競争心
- **劫財**: 同五行・異陰陽 → 行動力、短気

### 表現の星
- **食神**: 生じる五行・同陰陽 → 楽しみ、表現、円満
- **傷官**: 生じる五行・異陰陽 → 才能、批判精神、反骨

### 財の星
- **正財**: 剋す五行・異陰陽 → 堅実、固定財産、妻
- **偏財**: 剋す五行・同陰陽 → 社交性、流動財産、愛人

### 実行の星
- **正官**: 剋される五行・異陰陽 → 責任感、名誉、夫
- **偏官(七殺)**: 剋される五行・同陰陽 → 行動力、権力、闘争

### 知性の星
- **印綬**: 生じられる五行・異陰陽 → 学問、名誉、母
- **偏印(倒食)**: 生じられる五行・同陰陽 → 独創性、不安定、自由

## 十二運

日干が各地支で取るエネルギー状態(12段階):
長生 → 沐浴 → 冠帯 → 建禄 → 帝旺 → 衰 → 病 → 死 → 墓 → 絶 → 胎 → 養

- **帝旺**: 最強のエネルギー、支配力
- **建禄**: 独立、実力の発揮
- **絶**: 極限、危機と再生

## 日干の強弱判断

命式の中心である日干の強弱を判定:

### 判断要素
1. **月令を得ているか**: 生まれ月の季節が日干を支援するか
2. **通根しているか**: 地支の蔵干に日干と同じ五行があるか
3. **透干しているか**: 他の天干に日干と同じ五行が現れているか
4. **生助を得ているか**: 日干を生じる五行(印星)が多いか

### 結論
- **身強**: 日干を抑制・消耗させる五行(食傷・財・官殺)が喜神
- **身弱**: 日干を支援・強化する五行(印・比劫)が喜神

## 格局理論

### 内格(普通格局)
バランスを取ることが目的:
- 身強 → 食傷格、財格、官殺格(抑制が必要)
- 身弱 → 印綬格、比劫格(支援が必要)

### 外格(特別格局)
強い流れに従うことが目的:
- 従旺格: ほぼ比劫・印 → さらに強化が吉
- 従児格: 食傷が極強 → 食傷を伸ばす
- 従財格: 財星が極強 → 財を追求
- 従殺格: 官殺が極強 → 権力に従う

## 大運・年運

### 大運(10年運)
立運年齢の計算:
- 陽年男性・陰年女性: (次の節入り日 - 生まれ日) ÷ 3 (順行)
- 陰年男性・陽年女性: (生まれ日 - 前の節入り日) ÷ 3 (逆行)

### 運勢判断
巡ってきた大運・年運の五行が:
- **喜神** → 幸運期、物事がスムーズ
- **忌神** → 試練期、慎重な行動が必要

## 特殊な作用

### 干合・支合(調和)
- 天干の合化: 甲己合化土、乙庚合化金など
- 地支の合: 支合、三合局

### 冲・刑・害・破(不調和)
- **冲**: 子午、丑未など正反対の支 → 急変、対立
- **刑**: 寅巳申、丑戌未など → 葛藤、裏切り
- **害**: 支合を妨害する組み合わせ → 嫉妬、妨害
- **破**: 約束の破綻、信頼崩壊

### 空亡(天中殺)
12支のうち2支が「余り」となる期間:
- 新規開始は避ける
- 内省・整理の時期として活用

## 神殺

### 主要な吉神
- **天乙貴人**: 最強の守護神、困難からの救済
- **天徳貴人**: 物質的援助、穏やかな運
- **月徳貴人**: 精神的支援、人徳

### 主要な凶神
- **羊刃**: 極端なエネルギー、諸刃の剣
- **桃花(咸池)**: 異性関係、魅力と色難
- **駅馬**: 変動、移動、不安定
