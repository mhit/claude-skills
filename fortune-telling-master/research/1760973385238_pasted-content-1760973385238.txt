## エグゼクティブサマリー

6つの専門研究チームによる並行調査を実施し、四柱推命の自動鑑定システム構築に必要な情報を収集しました。**重要な発見として、「34-55歳ハイライト期間」は古典四柱推命の確立された教義ではなく現代的解釈であること、「アンダーテンション」は伝統的用語ではなく十二長生の弱運期に相当することが判明しました**。各調査項目について実装可能な形式で詳細をまとめています。

## 【調査項目2】34-55歳ハイライト期間の理論的根拠（最優先）

### 重要な発見

**「34-55歳がハイライト期間」という概念は、古典四柱推命の文献に明確に定義された普遍的ルールではありません**。これは以下の要素を統合した現代的解釈です：

1. 日柱の年齢域（33-48歳）
2. 大運の好運期が中年に訪れる理想的パターン
3. 社会的キャリアピーク年齢の統計的観察

### 理論的根拠

**四柱年齢域理論に基づく**：
- 年柱: 1-16歳、月柱: 17-32歳
- **日柱: 33-48歳（壮年期・人生の最盛期）**
- 時柱: 49歳以降

**34歳が特別な理由**: 年柱と月柱の衝突による事故の可能性は特に34歳以前に高く、34歳は月柱の不安定期から日柱の安定期への移行点を示します

**大運理論**: 清朝学者陳素菴「良い大運は10-20年、長くても30-40年だが、中年に良い大運が訪れるのが最も良い」

### 判定方法

```
ステップ1: 大運の開始年齢を計算
ステップ2: 日柱の質を評価（身強・身弱）
ステップ3: 各年齢に大運周期をマッピング
ステップ4: 大運の質を評価
ステップ5: ハイライト期間を算出
  = 日柱年齢域（33-48歳）× 好運の大運周期 × 日干の強弱
```

### 個人差の要因

- **要因A: 日干の強弱（40%影響）** - 身強は食傷・財星の大運で、身弱は印星・比劫の大運でハイライト
- **要因B: 大運の通変星（30%影響）** - 正官・偏官でキャリアピーク、財星で財運成功
- **要因C: 特殊格局（20%影響）** - 従格・化格・専旺格は異なるルール
- **要因D: 十二長生の位置（10%影響）** - 帝旺・建禄で延長、衰・病・死で短縮

### 実装アルゴリズム

```pseudocode
FUNCTION calculate_highlight_period(birth_date, birth_time, gender):
    chart = generate_bazi_chart(birth_date, birth_time)
    dayun_cycles = generate_dayun_cycles(chart, gender)
    dm_strength = calculate_day_master_strength(chart)
    
    highlight_scores = {}
    FOR age = 20 TO 70:
        score = 0
        IF age >= 33 AND age <= 48: score += 40
        ELSE IF age >= 34 AND age <= 55: score += 30
        
        current_dayun = get_dayun_for_age(dayun_cycles, age)
        IF current_dayun.element IN favorable_elements: score += 30
        
        score += evaluate_ten_god_favorability(current_dayun, chart)
        highlight_scores[age] = score
    
    threshold = 65
    highlight_years = [age FOR age IF score >= threshold]
    RETURN HighlightPeriod(MIN(highlight_years) OR 34, MAX(highlight_years) OR 55)
```

### 実装推奨

34-55をハードコードせず、個人の命式に基づいて好運大運期間を計算し、日柱域との重複を特定してカスタマイズされた盛期を提示してください。

### 重要な発見

**「アンダーテンション」という用語は伝統的な四柱推命文献に標準的な用語として存在しません**。これは現代的または専門的な用語のようです。

### 伝統的BaZiにおける対応概念

「弱運期間」や「エネルギー低下」は**十二長生（十二運）**システムで表現されます。

**「アンダーテンション」= 弱運段階**: 衰、病、死、墓、絶

### 理論的根拠

**五行相生相剋理論**: 各天干は季節によって強弱が変わる
- 木は春に強く、秋（金が剋す）に弱い
- 火は夏に強く、冬（水が剋す）に弱い
- 金は秋に強く、夏（火が剋す）に弱い
- 水は冬に強く、晩夏（土が剋す）に弱い

**季節関係（旺相休囚死）**: 旺＞相＞休＞囚（弱）＞死（最弱）

**陽生陰死の原理**: 陽干は順行、陰干は逆行。陽干が「死」の時、陰干は同じ支で「長生」
